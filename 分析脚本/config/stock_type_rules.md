# 股票类型识别规则

> **文档版本**: 2026-03-15
> **用途**: 阶段1使用，用于股票类型识别
> **更新说明**: 新增前瞻性指标、国产替代加成、行业阶段动态阈值

---

## 三层识别流程

```
第一层：行业属性判断 + 行业阶段判断 + 国产替代属性判断
    ↓
第二层：财务指标验证（含高成长赛道保护逻辑）
    ↓
第三层：潜力指标验证（仅对潜力成长股）
```

---

## 第一层：行业属性判断 + 行业阶段判断

```
【行业属性判断】
IF 公司所属行业 ∈ 高成长赛道清单
    → 行业属性 = "高成长赛道"
ELSE
    → 行业属性 = "普通行业"

【行业阶段判断】（仅对高成长赛道）
IF 渗透率<5% AND (技术处于验证期 OR 市场规模<1000亿)
    → 行业阶段 = "起步期"
ELSE IF 渗透率5%-20% OR (技术规模化应用 AND 政策体系完善)
    → 行业阶段 = "成长期"
ELSE
    → 行业阶段 = "成熟期"

【国产替代属性判断】（仅对高成长赛道）
IF 行业 ∈ 国产替代重点行业清单
   AND 国产化率 < 30%
   AND 公司是主要国产替代者（份额前5或头部客户验证）
    → 国产替代属性 = true
ELSE
    → 国产替代属性 = false
```

**国产替代重点行业清单**：
- 半导体设备、半导体材料、EDA软件
- 高端机床、工业机器人、伺服系统
- 航空发动机、高端医疗器械
- 操作系统、数据库、工业软件

---

## 第二层：财务指标验证（决策树）

### 前置检查：高成长赛道保护逻辑

```
【高成长赛道保护】（在转型股识别前执行）
IF 行业属性 = "高成长赛道"
   AND 主营业务稳定（无明确转型战略声明）
   AND 新业务占比 < 10%
    → 跳过转型股识别，直接进入成长类型判断
```

### 完整决策树

```
【优先级1：亏损股】
IF 净利润（近3年平均）< 0
    → 最终类型 = 亏损股

【优先级2：周期股】
ELSE IF 利润增速标准差 > 行业平均 × 2
    → 最终类型 = 周期股

【优先级2.5：高成长赛道保护】（新增）
ELSE IF 行业属性 = "高成长赛道"
        AND 主营业务稳定
        AND 新业务占比 < 10%
    → 跳过转型股识别，进入【优先级4-7】

【优先级3：转型股/早期转型股】
ELSE IF 满足转型股或早期转型股条件
    IF 新业务所处行业阶段 = "起步期"
        → 最终类型 = 早期转型股
    ELSE
        → 最终类型 = 转型股

【优先级3.5：潜力转型股】
ELSE IF 满足以下2条中任意1条：
    - 新业务营收占比 3%-10% 且 新业务增速 >100%
    - 新业务营收占比 >3% 且 新业务所在行业CAGR >15%
    → 最终类型 = 潜力转型股

【优先级4：早期成长股】（行业阶段动态阈值）
ELSE IF 行业属性 = "高成长赛道" AND 行业阶段 = "起步期"
    IF 营收CAGR > 5% AND (公司地位领先 OR 国产替代属性 = true)
        → 最终类型 = 早期成长股
    ELSE IF 营收CAGR > -5% AND 满足潜力指标(至少1个)
        → 最终类型 = 潜力成长股
    ELSE
        → 最终类型 = 成熟股

【优先级5-7：成长类型判断】（行业阶段动态阈值）
ELSE IF 行业属性 = "高成长赛道"

    【起步期】
    IF 行业阶段 = "起步期"
        IF 营收CAGR > 50%
            → 最终类型 = 高成长股
        ELSE IF 营收CAGR > 5%
            → 最终类型 = 成长股
        ELSE IF 营收CAGR > -5%
            → 候选"潜力成长股"（进入第三层，需满足1个潜力指标）
        ELSE
            → 最终类型 = 成熟股

    【成长期】
    ELSE IF 行业阶段 = "成长期"
        IF 营收CAGR > 50%
            → 最终类型 = 高成长股
        ELSE IF 营收CAGR > 20%
            → 最终类型 = 成长股
        ELSE IF 营收CAGR > 0%
            → 候选"潜力成长股"（进入第三层，需满足1个潜力指标）
        ELSE
            → 最终类型 = 成熟股

    【成熟期】
    ELSE（成熟期）
        IF 营收CAGR > 50%
            → 最终类型 = 高成长股
        ELSE IF 营收CAGR > 20%
            → 最终类型 = 成长股
        ELSE IF 营收CAGR > 0%
            → 候选"潜力成长股"（进入第三层，需满足2个潜力指标）
        ELSE
            → 最终类型 = 成熟股

ELSE（普通行业）
    IF 营收CAGR > 50%
        → 最终类型 = 高成长股
    ELSE IF 营收CAGR > 15%
        → 最终类型 = 成长股
    ELSE
        → 最终类型 = 成熟股
```

## 第三层：潜力指标验证（仅对"潜力成长股"）

**触发条件**：
- 高成长赛道 + 营收CAGR在对应行业阶段阈值范围内
- 起步期：CAGR -5%~5%
- 成长期：CAGR 0%~20%
- 成熟期：CAGR 0%~20%

**潜力指标清单（10个，满足指定数量即可）**：

| # | 潜力指标 | 指标说明 | 基础阈值 | 增强阈值 | 爆发阈值 |
|---|:---|:---|:---|:---|:---|
| 1 | **在手订单覆盖** | 订单储备/年收入比 | >100% | >150% | >200% |
| 2 | **行业渗透率** | 增长空间+竞争地位 | <20%且头部 | <10%且头部 | <5%且龙头 |
| 3 | **技术储备深度** | 研发投入+技术壁垒 | 研发>5%或有专利 | 研发>8%或核心技术 | 研发>10%或独家技术 |
| 4 | **客户质量提升** | 市场认可度 | 头部客户>30% | 头部客户>50% | 全球顶级客户 |
| 5 | **产能就绪度** | 产能储备/当前产能 | >1.2倍 | >1.5倍 | >2倍 |
| 6 | **核心客户突破** | 进入头部客户供应链 | 已进入1家头部客户 | 已进入2家头部客户 | 已进入3家+头部客户 |
| 7 | **产品认证进展** | 关键产品认证状态 | 已通过或即将通过关键认证 | 通过车规/军工认证 | 通过国际顶级客户认证 |
| 8 | **产能建设进度** | 新产能建设状态 | 新产能已建成 | 新产能建成+试产爬坡 | 新产能满产+继续扩产 |
| 9 | **国产替代进程** | 国产替代中的地位 | 国产化率<50%且主要替代者 | 国产化率<30%且核心替代者 | 国产化率<20%且独家替代者 |
| 10 | **管理层指引** | 公司增长预期 | 管理层给出>20%增长指引 | 管理层给出>30%增长指引 | 管理层给出>50%增长指引 |

**头部客户定义**：
- 半导体：华为、中芯国际、长江存储、长鑫存储
- 新能源：特斯拉、宁德时代、比亚迪
- 消费电子：苹果、华为、小米
- 其他行业：行业前3企业

**潜力指标满足数量要求（按行业阶段）**：

| 行业阶段 | 需满足指标数量 | 说明 |
| :--- | :--- | :--- |
| **起步期** | 1个 | 早期数据不全，门槛降低 |
| **成长期** | 1个 | 维持原有标准 |
| **成熟期** | 2个 | 成熟期需要更强验证 |

**判断逻辑**：
```
满足指标数量 = 潜力指标满足数量
required_count = 按行业阶段要求的指标数量

IF 国产替代属性 = true AND 已有头部客户验证
    → required_count = required_count - 1（国产替代加成）

IF 满足指标数量 >= required_count
    → 最终类型 = 潜力成长股
ELSE
    → 最终类型 = 成熟股
```

## 国产替代加成规则

**加成条件**（必须同时满足）：
1. 行业 ∈ 国产替代重点行业清单
2. 国产化率 < 30%（仅针对国产替代早期阶段）
3. 公司是主要国产替代者（市场份额前5或已有头部客户验证）
4. 已有头部客户验证（华为/中芯/长存/特斯拉/宁德等）

**加成效果**：
```
IF 满足国产替代加成条件:
    - 潜力成长股识别时，指标数量要求 -1（最低为1）
    - 成长系数上限 +0.2（在阶段2评分中体现）
    - 在报告"核心逻辑"中体现国产替代主题
```

## 股票类型汇总表（10种类型）

| 股票类型 | 识别条件 | 特征描述 |
| :--- | :--- | :--- |
| **早期成长股** | 高成长赛道起步期 + 增速>5% + (领先地位 OR 国产替代) | 行业早期，增长受基数限制 |
| **早期转型股** | 转向起步期行业 + 新业务占比>3% + 增速>30% + 转型验证 | 转型方向处于早期 |
| **高成长股** | 营收增速>50% | 处于业绩爆发期，高速扩张 |
| **成长股** | 营收增速在行业阶段对应阈值范围内 | 成长动能强劲 |
| **潜力成长股** | 高成长赛道 + CAGR在阈值内 + 潜力指标支撑（按阶段要求） | 处于产能爬坡期，未来增长可期 |
| **成熟股** | 增速低于行业阶段阈值 | 稳定经营，增长缓慢 |
| **周期股** | 近3年利润增速标准差 > 行业平均 × 2 | 盈利随周期波动 |
| **亏损股** | 净利润（近3年平均）< 0 | 处于投入期，尚未盈利 |
| **转型股** | 满足以下任意2条：<br>1. 新业务营收占比>5% 且增速>50%<br>2. 新业务占比年度提升>3个百分点<br>3. 新业务所在行业CAGR>15%<br>4. 研发占比>8% 且新方向投入>50%<br>5. 管理层明确宣布转型战略 | 从传统业务向高成长赛道转型 |
| **潜力转型股** | 满足以下2条中任意1条：<br>1. 新业务营收占比3%-10% 且增速>100%<br>2. 新业务营收占比>3% 且所在行业CAGR>15% | 早期转型，新业务高增长 |

**识别优先级**：亏损股 → 周期股 → 高成长赛道保护检查 → 早期转型股 → 转型股 → 潜力转型股 → 早期成长股 → 高成长股 → 成长股 → 潜力成长股 → 成熟股

| :--- | :--- | :--- | :--- | :--- |
| **起步期** | >50% | >5% | -5%~5% | 1个 |
| **成长期** | >50% | >20% | 0%~20% | 1个 |
| **成熟期** | >50% | >20% | 0%~20% | 2个 |

## 早期成长股详细定义

**定义**：身处高成长行业的**起步阶段**，公司增速受行业基数限制，但表现优于行业平均水平，或处于国产替代关键位置。

**识别条件**（必须同时满足）：
1. 所属行业 ∈ 高成长赛道
2. 行业阶段 = "起步期"（渗透率<5% OR 技术验证期）
3. 公司营收CAGR > 5%（原10%已下调）
4. 公司在行业中具有**领先地位或国产替代属性**（满足以下任一）：
   - 市场份额前5
   - 已获得头部客户验证（如华为、中芯国际、特斯拉等）
   - 拥有核心专利或独家技术
   - **国产替代属性 = true**

---

## 早期转型股详细定义

**定义**：从传统行业转向**起步阶段**的高成长赛道，新业务增速受行业基数限制。

**识别条件**（必须同时满足）：
1. 新业务方向 ∈ 高成长赛道
2. 新业务所处行业阶段 = "起步期"
3. 新业务营收占比 > 3%
4. 新业务增速 > 30%
5. **转型验证**（满足以下任一）：
   - 新业务已获得头部客户订单
   - 新业务拥有核心技术/专利
   - 管理层明确宣布转型战略且有投入（资本开支或研发投入）

## 类型识别所需数据

| 识别层级 | 识别条件 | 所需数据 |
|---------|---------|---------|
| 第一层 | 行业属性判断 | 所属行业、主营业务描述 |
| 第一层 | 行业阶段判断 | 渗透率、技术成熟度、市场规模、政策支持度 |
| 第一层 | 国产替代属性判断 | 国产化率、公司替代者地位、头部客户验证 |
| 第二层-亏损股 | 净利润<0 | 净利润近3年数据 |
| 第二层-周期股 | 利润增速标准差>行业平均×2 | 近3年利润增速标准差 |
| 第二层-高成长保护 | 主营业务稳定+新业务<10% | 转型战略声明、新业务占比 |
| 第二层-转型股 | 满足转型股条件 | 新业务相关数据（5项） |
| 第二层-早期成长股 | 起步期+增速>5%+(领先OR国产替代) | 公司市场份额、头部客户验证、专利/核心技术、国产替代属性 |
| 第二层-早期转型股 | 转向起步期行业 | 新业务行业阶段、新业务占比、新业务增速 |
| 第二层-成长股 | 营收CAGR判断（按行业阶段） | 营业收入近3年数据、行业阶段 |
| 第三层-潜力成长股 | 潜力指标验证（10个指标） | 在手订单、产能就绪度、技术储备、客户质量、核心客户突破、产品认证、产能建设、国产替代进程、管理层指引等 |
