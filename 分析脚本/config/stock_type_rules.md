# 股票类型识别规则

> **文档版本**: 2026-03-14
> **用途**: 阶段1使用，用于股票类型识别

---

## 三层识别流程

```
第一层：行业属性判断 + 行业阶段判断
    ↓
第二层：财务指标验证（含早期类型判断）
    ↓
第三层：潜力指标验证（仅对潜力成长股）
```

---

## 第一层：行业属性判断 + 行业阶段判断

```
【行业属性判断】
IF 公司所属行业 ∈ 高成长赛道清单
    → 行业属性 = "高成长赛道"
ELSE
    → 行业属性 = "普通行业"

【行业阶段判断】（仅对高成长赛道）
IF 渗透率<5% AND (技术处于验证期 OR 市场规模<1000亿)
    → 行业阶段 = "起步期"
ELSE IF 渗透率5%-20% OR (技术规模化应用 AND 政策体系完善)
    → 行业阶段 = "成长期"
ELSE
    → 行业阶段 = "成熟期"
```

---

## 第二层：财务指标验证（决策树）

```
【优先级1：亏损股】
IF 净利润（近3年平均）< 0
    → 最终类型 = 亏损股

【优先级2：周期股】
ELSE IF 利润增速标准差 > 行业平均 × 2
    → 最终类型 = 周期股

【优先级3：转型股/早期转型股】
ELSE IF 满足转型股或早期转型股条件
    IF 新业务所处行业阶段 = "起步期"
        → 最终类型 = 早期转型股
    ELSE
        → 最终类型 = 转型股

【优先级3.5：潜力转型股】
ELSE IF 满足以下2条中任意1条：
    - 新业务营收占比 3%-10% 且 新业务增速 >100%
    - 新业务营收占比 >3% 且 新业务所在行业CAGR >15%
    → 最终类型 = 潜力转型股

【优先级4：早期成长股】
ELSE IF 行业属性 = "高成长赛道" AND 行业阶段 = "起步期"
    IF 营收增速 > 10% AND 公司地位领先（份额前5 或 头部客户验证）
        → 最终类型 = 早期成长股
    ELSE IF 营收增速 > 0% AND 潜力指标支撑
        → 最终类型 = 潜力成长股
    ELSE
        → 最终类型 = 成熟股

【优先级5-7：成长类型判断】
ELSE IF 行业属性 = "高成长赛道"
    IF 营收CAGR > 50%
        → 最终类型 = 高成长股
    ELSE IF 营收CAGR > 20%
        → 最终类型 = 成长股
    ELSE IF 营收CAGR > 0%
        → 候选"潜力成长股"（进入第三层）
    ELSE
        → 最终类型 = 成熟股

ELSE（普通行业）
    IF 营收CAGR > 50%
        → 最终类型 = 高成长股
    ELSE IF 营收CAGR > 15%
        → 最终类型 = 成长股
    ELSE
        → 最终类型 = 成熟股
```

## 第三层：潜力指标验证（仅对"潜力成长股"）

**触发条件**：高成长赛道 + 营收CAGR 0%-20%

**潜力指标清单**（满足任一即可）：

| 潜力指标 | 指标说明 | 基础阈值 | 增强阈值 | 爆发阈值 |
| :--- | :--- | :--- | :--- | :--- |
| 在手订单覆盖 | 订单储备/年收入比 | >100% | >150% | >200% |
| 行业渗透率 | 增长空间+竞争地位 | <20%且头部 | <10%且头部 | <5%且龙头 |
| 技术储备深度 | 研发投入+技术壁垒 | 研发>5%或有专利 | 研发>8%或核心技术 | 研发>10%或独家技术 |
| 客户质量提升 | 市场认可度 | 头部客户>30% | 头部客户>50% | 全球顶级客户 |
| 产能就绪度 | 产能储备/当前产能 | >1.2倍 | >1.5倍 | >2倍 |

**判断逻辑**：
```
IF 满足任一潜力指标
    → 最终类型 = 潜力成长股
ELSE
    → 最终类型 = 成熟股
```

## 股票类型汇总表（10种类型）

| 股票类型 | 识别条件 | 特征描述 |
| :--- | :--- | :--- |
| **早期成长股** | 高成长赛道起步期 + 增速>10% + 领先地位（份额前5或头部客户验证） | 行业早期，增长受基数限制 |
| **早期转型股** | 转向起步期行业 + 新业务占比>3% + 增速>30% + 转型验证 | 转型方向处于早期 |
| **高成长股** | 营收增速>50% | 处于业绩爆发期，高速扩张 |
| **成长股** | 营收增速20%-50%（高成长赛道）或 15%-50%（普通行业） | 成长动能强劲 |
| **潜力成长股** | 高成长赛道 + 营收增速0%-20% + 潜力指标支撑 | 处于产能爬坡期，未来增长可期 |
| **成熟股** | 增速<20%（高成长赛道）或 <15%（普通行业） | 稳定经营，增长缓慢 |
| **周期股** | 近3年利润增速标准差 > 行业平均 × 2 | 盈利随周期波动 |
| **亏损股** | 净利润（近3年平均）< 0 | 处于投入期，尚未盈利 |
| **转型股** | 满足以下任意2条：<br>1. 新业务营收占比>5% 且增速>50%<br>2. 新业务占比年度提升>3个百分点<br>3. 新业务所在行业CAGR>15%<br>4. 研发占比>8% 且新方向投入>50%<br>5. 管理层明确宣布转型战略 | 从传统业务向高成长赛道转型 |
| **潜力转型股** | 满足以下2条中任意1条：<br>1. 新业务营收占比3%-10% 且增速>100%<br>2. 新业务营收占比>3% 且所在行业CAGR>15% | 早期转型，新业务高增长 |

**识别优先级**：亏损股 → 周期股 → 早期转型股 → 转型股 → 潜力转型股 → 早期成长股 → 高成长股 → 成长股 → 潜力成长股 → 成熟股

---

## 早期成长股详细定义

**定义**：身处高成长行业的**起步阶段**，公司增速受行业基数限制，但表现优于行业平均水平。

**识别条件**（必须同时满足）：
1. 所属行业 ∈ 高成长赛道
2. 行业阶段 = "起步期"（渗透率<5% OR 技术验证期）
3. 公司营收增速 > 10%
4. 公司在行业中具有**领先地位**（满足以下任一）：
   - 市场份额前5
   - 已获得头部客户验证（如华为、特斯拉、OpenAI等）
   - 拥有核心专利或独家技术

---

## 早期转型股详细定义

**定义**：从传统行业转向**起步阶段**的高成长赛道，新业务增速受行业基数限制。

**识别条件**（必须同时满足）：
1. 新业务方向 ∈ 高成长赛道
2. 新业务所处行业阶段 = "起步期"
3. 新业务营收占比 > 3%
4. 新业务增速 > 30%
5. **转型验证**（满足以下任一）：
   - 新业务已获得头部客户订单
   - 新业务拥有核心技术/专利
   - 管理层明确宣布转型战略且有投入（资本开支或研发投入）

## 类型识别所需数据

| 识别层级 | 识别条件 | 所需数据 |
|---------|---------|---------|
| 第一层 | 行业属性判断 | 所属行业、主营业务描述 |
| 第一层 | 行业阶段判断 | 渗透率、技术成熟度、市场规模、政策支持度 |
| 第二层-亏损股 | 净利润<0 | 净利润近3年数据 |
| 第二层-周期股 | 利润增速标准差>行业平均×2 | 近3年利润增速标准差 |
| 第二层-转型股 | 满足转型股条件 | 新业务相关数据（5项） |
| 第二层-早期成长股 | 起步期+增速>10%+领先地位 | 公司市场份额、头部客户验证、专利/核心技术 |
| 第二层-早期转型股 | 转向起步期行业 | 新业务行业阶段、新业务占比、新业务增速 |
| 第二层-成长股 | 营收CAGR判断 | 营业收入近3年数据 |
| 第三层-潜力成长股 | 潜力指标验证 | 在手订单、产能就绪度、技术储备、客户质量等 |
