# 股票类型识别规则（精简版）

> **用途**: 阶段1使用，用于股票类型识别
> **完整版**: [stock_type_rules.md](./stock_type_rules.md)
> **更新说明**: 新增前瞻性指标、国产替代加成、行业阶段动态阈值

---

## 三层识别流程

```
第一层：行业属性判断 + 行业阶段判断 + 国产替代属性判断
    ↓
第二层：财务指标验证（含高成长赛道保护逻辑）
    ↓
第三层：潜力指标验证（仅对潜力成长股，按行业阶段要求）
```

---

## 第一层：行业属性判断 + 行业阶段判断

```
【行业属性判断】
IF 公司所属行业 ∈ 高成长赛道清单
    → 行业属性 = "高成长赛道"
ELSE
    → 行业属性 = "普通行业"

【行业阶段判断】（仅对高成长赛道）
IF 渗透率<5% AND (技术处于验证期 OR 市场规模<1000亿)
    → 行业阶段 = "起步期"
ELSE IF 渗透率5%-20% OR (技术规模化应用 AND 政策体系完善)
    → 行业阶段 = "成长期"
ELSE
    → 行业阶段 = "成熟期"

【国产替代属性判断】（仅对高成长赛道）
IF 行业 ∈ 国产替代重点行业清单
   AND 国产化率 < 30%
   AND 公司是主要国产替代者（份额前5或头部客户验证）
    → 国产替代属性 = true
ELSE
    → 国产替代属性 = false
```

**国产替代重点行业**: 半导体设备/材料、EDA、高端机床、工业机器人、航空发动机、高端医疗器械、操作系统、数据库、工业软件

---

## 第二层：财务指标验证（决策树）

```
【优先级1：亏损股】
IF 净利润（近3年平均）< 0
    → 最终类型 = 亏损股

【优先级2：周期股】
ELSE IF 利润增速标准差 > 行业平均 × 2
    → 最终类型 = 周期股

【优先级2.5：高成长赛道保护】（新增）
ELSE IF 行业属性 = "高成长赛道"
        AND 主营业务稳定
        AND 新业务占比 < 10%
    → 跳过转型股识别，进入成长类型判断

【优先级3：转型股/早期转型股】
ELSE IF 满足转型股条件
    IF 新业务所处行业阶段 = "起步期"
        → 最终类型 = 早期转型股
    ELSE
        → 最终类型 = 转型股

【优先级3.5：潜力转型股】
ELSE IF 新业务占比3%-10%且增速>100% 或 新业务占比>3%且行业CAGR>15%
    → 最终类型 = 潜力转型股

【优先级4：早期成长股】
ELSE IF 行业属性 = "高成长赛道" AND 行业阶段 = "起步期"
    IF 营收CAGR > 5% AND (公司地位领先 OR 国产替代属性 = true)
        → 最终类型 = 早期成长股
    ELSE IF 营收CAGR > -5% AND 满足潜力指标(≥1个)
        → 最终类型 = 潜力成长股
    ELSE
        → 最终类型 = 成熟股

【优先级5-7：成长类型判断】（按行业阶段动态阈值）
ELSE IF 行业属性 = "高成长赛道"
    【起步期】IF CAGR>50%→高成长股, >5%→成长股, >-5%→候选潜力成长股
    【成长期】IF CAGR>50%→高成长股, >20%→成长股, >0%→候选潜力成长股
    【成熟期】IF CAGR>50%→高成长股, >20%→成长股, >0%→候选潜力成长股(需2个指标)

ELSE（普通行业）
    IF CAGR > 50% → 高成长股
    ELSE IF CAGR > 15% → 成长股
    ELSE → 成熟股
```

## 第三层：潜力指标验证（10个指标）

**触发条件**: 高成长赛道 + 营收CAGR在对应行业阶段阈值范围内

| # | 潜力指标 | 基础阈值 | 增强阈值 | 爆发阈值 |
|---|:---|:---|:---|:---|
| 1 | 在手订单覆盖 | >100% | >150% | >200% |
| 2 | 行业渗透率 | <20%且头部 | <10%且头部 | <5%且龙头 |
| 3 | 技术储备深度 | 研发>5%或有专利 | 研发>8%或核心技术 | 研发>10%或独家技术 |
| 4 | 客户质量提升 | 头部客户>30% | 头部客户>50% | 全球顶级客户 |
| 5 | 产能就绪度 | >1.2倍 | >1.5倍 | >2倍 |
| 6 | **核心客户突破** | 1家头部客户 | 2家头部客户 | 3家+头部客户 |
| 7 | **产品认证进展** | 已通过关键认证 | 车规/军工认证 | 国际顶级认证 |
| 8 | **产能建设进度** | 新产能已建成 | 建成+试产爬坡 | 满产+继续扩产 |
| 9 | **国产替代进程** | 国产化率<50%且主要替代者 | <30%且核心替代者 | <20%且独家替代者 |
| 10 | **管理层指引** | 增长>20%指引 | 增长>30%指引 | 增长>50%指引 |

**潜力指标要求（按行业阶段）**:
- 起步期：1个
- 成长期：1个
- 成熟期：2个

**国产替代加成**: IF 国产替代属性=true AND 头部客户验证 → 指标数量要求-1（最低1）

```
IF 满足指标数量 >= 要求数量
    → 最终类型 = 潜力成长股
ELSE
    → 最终类型 = 成熟股
```

## 股票类型汇总表（10种类型）

| 股票类型 | 识别条件 |
| :--- | :--- |
| **早期成长股** | 高成长赛道起步期 + 增速>5% + (领先地位 OR 国产替代) |
| **早期转型股** | 转向起步期行业 + 新业务占比>3% + 增速>30% + 转型验证 |
| **高成长股** | 营收增速>50% |
| **成长股** | 营收增速在行业阶段对应阈值范围内 |
| **潜力成长股** | 高成长赛道 + CAGR在阈值内 + 潜力指标支撑（按阶段要求） |
| **成熟股** | 增速低于行业阶段阈值 |
| **周期股** | 近3年利润增速标准差 > 行业平均 × 2 |
| **亏损股** | 净利润（近3年平均）< 0 |
| **转型股** | 满足任意2条：新业务占比>5%且增速>50% / 占比年度提升>3% / 行业CAGR>15% / 研发>8%且新方向>50% / 管理层转型战略 |
| **潜力转型股** | 新业务占比3%-10%且增速>100% 或 新业务占比>3%且行业CAGR>15% |

**识别优先级**：亏损股 → 周期股 → 高成长赛道保护检查 → 早期转型股 → 转型股 → 潜力转型股 → 早期成长股 → 高成长股 → 成长股 → 潜力成长股 → 成熟股
