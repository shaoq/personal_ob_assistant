# 成长系数档位表（精简版）

> **用途**: 阶段2使用，用于计算最终综合评分
> **完整版**: [growth_coefficient.md](./growth_coefficient.md)

---

## 成长系数机制

```
最终综合评分 = min(基础综合评分 × 成长系数, 5.0)
```

**适用范围**：早期成长股、早期转型股、高成长股、转型股、潜力转型股、潜力成长股
**其他类型**：成长系数=1.00（不调整）

---

## 早期成长股成长系数

| 增速范围 | 公司地位 | 成长系数 |
|:---|:---|:---|
| 10%-20% | 行业龙头（份额第1） | 1.08 |
| 10%-20% | 行业领先（份额前5） | 1.05 |
| 20%-50% | 任意 | 1.10 |
| >50% | 任意 | 1.15 |

---

## 早期转型股成长系数

| 新业务增速 | 新业务占比 | 成长系数 |
|:---|:---|:---|
| 30%-50% | >5% | 1.05 |
| 50%-100% | >5% | 1.08 |
| >100% | >5% | 1.10 |
| 任意 | <5% | 1.00 |

---

## 高成长股成长系数

| 整体增速范围 | 成长系数 |
| :--- | :--- |
| 50%-80% | 1.00 |
| 80%-120% | 1.05 |
| 120%-200% | 1.10 |
| >200% | 1.15 |

---

## 转型股成长系数

| 新业务增速 | 新业务占比 | 成长系数 |
| :--- | :--- | :--- |
| 50%-100% | 任意 | 1.00 |
| 100%-150% | >10% | 1.05 |
| 150%-200% | >10% | 1.10 |
| **>200%** | **>15%** | **1.15** |
| >200% | 10%-15% | 1.10 |
| >100% | <10% | 1.00 |

## 潜力转型股成长系数（上限1.10）

| 新业务增速 | 新业务占比 | 成长系数 |
| :--- | :--- | :--- |
| 100%-150% | 任意 | 1.05 |
| 150%-200% | >5% | 1.08 |
| **>200%** | **>5%** | **1.10** |

---

## 潜力成长股成长系数

| 潜力强度 | 判断标准 | 成长系数 |
| :--- | :--- | :--- |
| **基础档** | 仅满足1个潜力指标 | 1.00 |
| **增强档** | 满足2个潜力指标或单个达到增强阈值 | 1.05 |
| **爆发档** | 满足3个及以上或单个达到爆发阈值 | 1.10 |

**潜力指标阈值**：

| 潜力指标 | 基础阈值 | 增强阈值 | 爆发阈值 |
| :--- | :--- | :--- | :--- |
| 在手订单覆盖 | >100% | >150% | >200% |
| 行业渗透率 | <20%且头部 | <10%且头部 | <5%且龙头 |
| 技术储备深度 | 研发>5%或有专利 | 研发>8%或核心技术 | 研发>10%或独家技术 |
| 客户质量提升 | 头部客户>30% | 头部客户>50% | 全球顶级客户 |
| 产能就绪度 | >1.2倍 | >1.5倍 | >2倍 |

---

## 成长系数上限约束

- **高成长股/转型股**：最高1.15
- **潜力成长股/潜力转型股**：最高1.10
- 最终综合评分不得超过5.0分
