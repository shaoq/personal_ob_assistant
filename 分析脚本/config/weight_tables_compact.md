## 转型股行业前景"双轨制"评分

```
行业前景综合评分 = 原主业评分 × 原主业权重 + 新业务评分 × 新业务权重
```

**权重确定原则**：

| 转型阶段 | 新业务占比 | 原主业权重 | 新业务权重 |
| :--- | :--- | :--- | :--- |
| **潜力转型股** | 3%-10% | 60% | 40% |
| 早期（新业务占比<10%） | <10% | 70% | 30% |
| 中期（新业务占比10%-30%） | 10%-30% | 50% | 50% |
| 后期（新业务占比>30%） | >30% | 30% | 70% |
