# 成长系数档位表

> **文档版本**: 2026-03-14
> **用途**: 阶段2使用，用于计算最终综合评分

---

## 成长系数机制

### 设计目的

在现有动态权重评分体系基础上，通过**成长系数**来体现超高成长性的额外价值：

```
最终综合评分 = min(基础综合评分 × 成长系数, 5.0)
```

### 适用范围

成长系数**适用于以下股票类型**：
- **早期成长股**（起步期行业+增速>10%+领先地位）
- **早期转型股**（转向起步期行业）
- **高成长股**（整体增速>50%）
- **转型股**（新业务增速>50%）
- **潜力转型股**（新业务增速>100%）
- **潜力成长股**（潜力指标支撑）

其他股票类型（成熟股、周期股、亏损股）**不适用**成长系数，成长系数=1.00。

| 新业务增速 | 新业务占比 | 成长系数 | 说明 |
| :--- | :--- | :--- | :--- |
| 100%-150% | 任意 | 1.05 | 早期高成长转型 |
| 150%-200% | >5% | 1.08 | 超高成长转型 |
| **>200%** | **>5%** | **1.10** | **爆发式转型** |

### 潜力成长股成长系数

> **核心思想**：成长系数基于"潜力指标强度"而非"增速强度"

| 潜力强度 | 判断标准 | 成长系数 | 说明 |
| :--- | :--- | :--- | :--- |
| **基础档** | 仅满足1个潜力指标，或指标刚过阈值 | 1.00 | 基础潜力，不调整 |
| **增强档** | 满足2个潜力指标，或单个指标达到增强阈值 | 1.05 | 潜力较强 |
| **爆发档** | 满足3个及以上潜力指标，或单个指标达到爆发阈值 | 1.10 | 潜力极强，即将爆发 |

**潜力指标强度判定规则**：

| 潜力指标 | 指标说明 | 基础阈值 | 增强阈值 | 爆发阈值 |
| :--- | :--- | :--- | :--- | :--- |
| 在手订单覆盖 | 订单储备/年收入比 | >100% | >150% | >200% |
| 行业渗透率 | 增长空间+竞争地位 | <20%且头部 | <10%且头部 | <5%且龙头 |
| 技术储备深度 | 研发投入+技术壁垒 | 研发>5%或有专利 | 研发>8%或核心技术 | 研发>10%或独家技术 |
| 客户质量提升 | 市场认可度 | 头部客户>30% | 头部客户>50% | 全球顶级客户 |
| 产能就绪度 | 产能储备/当前产能 | >1.2倍 | >1.5倍 | >2倍 |

**强度判定逻辑**：
```
IF 满足3个及以上潜力指标
    → 强度 = "爆发档"（系数1.10）
ELSE IF 满足2个潜力指标
    → 强度 = "增强档"（系数1.05）
ELSE IF 满足1个潜力指标 且 该指标达到爆发阈值
    → 强度 = "爆发档"（系数1.10）
ELSE IF 满足1个潜力指标 且 该指标达到增强阈值
    → 强度 = "增强档"（系数1.05）
ELSE
    → 强度 = "基础档"（系数1.00）
```

---

## 成长系数上限约束

**重要约束**：
- **高成长股**和**转型股**：成长系数最高为 **1.15**
- **潜力成长股**：成长系数最高为 **1.10**（体现"潜力"而非"现实"）
- 最终综合评分**不得超过5.0分**（评级体系上限）
- 若基础评分 × 成长系数 > 5.0，则取 **5.0分**
