# 行业阶段定义

> **文档版本**: 2026-03-15
> **用途**: 阶段1使用，用于判断高成长赛道的行业阶段
> **更新说明**: 新增各阶段增速宽松度、国产替代相关内容

---

## 行业阶段分类

仅对**高成长赛道**进行行业阶段判断，普通行业不需要判断阶段。

| 行业阶段 | 渗透率 | 技术状态 | 市场规模 | 政策环境 | 典型行业 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **起步期** | <5% | 技术验证期 | <1000亿 | 政策框架初建 | 人形机器人、量子计算、脑机科学 |
| **成长期** | 5%-20% | 技术规模化应用 | 1000亿-5000亿 | 政策体系完善 | AI应用、数据中心、新能源车 |
| **成熟期** | >20% | 技术稳定迭代 | >5000亿 | 政策成熟/收紧 | 传统半导体、光伏、风电 |

---

## 行业阶段判断逻辑

```
IF 渗透率<5% AND (技术处于验证期 OR 市场规模<1000亿)
    → 行业阶段 = "起步期"
ELSE IF 渗透率5%-20% OR (技术规模化应用 AND 政策体系完善)
    → 行业阶段 = "成长期"
ELSE
    → 行业阶段 = "成熟期"
```

---

## 各阶段特征详解

### 起步期

**特征**：
- 渗透率 < 5%，市场空间巨大
- 技术处于验证期，路线可能分化
- 市场规模 < 1000亿，但增速极快
- 政策框架初建，补贴和标准待完善

**投资特点**：
- 高风险高回报
- 公司增速受行业基数限制（10%-30%即为优秀）
- 需关注技术路线、头部客户验证、先发优势

**典型行业**：
- 人形机器人（渗透率<1%）
- 量子计算（技术验证期）
- 脑机科学（早期研发阶段）

**增速宽松度**：
- 早期成长股增速门槛降至 **>5%**（原10%）
- 潜力成长股CAGR容忍短期负增长（**-5%~5%**）
- 潜力指标满足 **1个** 即可
- 国产替代进程可作为增强因子

**国产替代关注点**：
- 起步期行业通常国产化率极低（<10%）
- 重点评估：是否进入头部客户验证阶段
- 技术路线是否与国际同步或接近

---

### 成长期

**特征**：
- 渗透率 5%-20%，快速渗透
- 技术规模化应用，路线基本确定
- 市场规模 1000亿-5000亿
- 政策体系完善，标准明确

**投资特点**：
- 成长确定性高
- 公司增速通常 20%-50%
- 需关注市场份额、产能扩张、盈利能力

**典型行业**：
- AI应用（渗透率10%-15%）
- 数据中心（渗透率15%-20%）
- 新能源车（渗透率25%+）

**增速宽松度**：
- 早期成长股不适用（非起步期）
- 成长股门槛：**>20%**
- 潜力成长股CAGR：**0%~20%**
- 潜力指标满足 **1个** 即可

**国产替代关注点**：
- 成长期行业国产化率通常 10%-30%
- 重点评估：市场份额提升速度
- 是否具备规模量产能力

---

### 成熟期

**特征**：
- 渗透率 > 20%，增长放缓
- 技术稳定迭代，创新空间有限
- 市场规模 > 5000亿
- 政策成熟或开始收紧

**投资特点**：
- 行业增速放缓至 10%-20%
- 公司竞争格局稳定
- 需关注成本控制、分红能力、第二增长曲线

**典型行业**：
- 传统半导体（渗透率>30%）
- 光伏（渗透率>25%）
- 风电（渗透率>20%）

**增速宽松度**：
- 早期成长股不适用（非起步期）
- 成长股门槛：**>20%**（与成长期一致）
- 潜力成长股CAGR：**0%~20%**
- 潜力指标满足 **2个**（要求更高）

**国产替代关注点**：
- 成熟期行业国产化率通常 20%-50%
- 重点评估：在存量市场中的替代空间
- 是否具备技术迭代能力

## 行业阶段与股票类型对应关系

| 行业阶段 | 可能的股票类型 | 识别条件 | 潜力指标要求 |
| :--- | :--- | :--- | :--- |
| **起步期** | 早期成长股 | 增速>5% + (领先地位 OR 国产替代) | - |
| **起步期** | 潜力成长股 | CAGR -5%~5% + 潜力指标 | 1个 |
| **起步期** | 早期转型股 | 转向起步期行业 + 新业务占比>3% + 增速>30% | - |
| **成长期** | 高成长股 | 增速>50% | - |
| **成长期** | 成长股 | 增速20%-50% | - |
| **成长期** | 潜力成长股 | 增速0%-20% + 潜力指标 | 1个 |
| **成熟期** | 高成长股 | 增速>50% | - |
| **成熟期** | 成长股 | 增速20%-50% | - |
| **成熟期** | 潜力成长股 | 增速0%-20% + 潜力指标 | 2个 |
| **成熟期** | 成熟股 | 增速<20% | - |

## 行业阶段与增速阈值对照表

| 行业阶段 | 高成长股 | 成长股 | 潜力成长股CAGR | 潜力指标要求 |
| :--- | :--- | :--- | :--- | :--- |
| **起步期** | >50% | >5% | -5%~5% | 1个 |
| **成长期** | >50% | >20% | 0%~20% | 1个 |
| **成熟期** | >50% | >20% | 0%~20% | 2个 |

## 行业阶段数据来源

| 数据项 | 数据来源 | 用途 |
| :--- | :--- | :--- |
| 行业渗透率 | 行业研究报告、行业协会数据 | 判断行业阶段 |
| 技术成熟度 | 技术白皮书、专家观点 | 判断技术状态 |
| 市场规模 | 行业研究报告、统计数据 | 判断市场规模 |
| 政策环境 | 政策文件、监管动态 | 判断政策支持度 |
| 国产化率 | 行业研究报告、海关数据 | 判断国产替代空间 |
