# 行业阶段定义（精简版）

> **用途**: 阶段1使用，用于判断高成长赛道的行业阶段
> **完整版**: [industry_stages.md](./industry_stages.md)

---

## 行业阶段分类

| 行业阶段 | 渗透率 | 技术状态 | 市场规模 | 典型行业 |
| :--- | :--- | :--- | :--- | :--- |
| **起步期** | <5% | 技术验证期 | <1000亿 | 人形机器人、量子计算、脑机科学 |
| **成长期** | 5%-20% | 技术规模化应用 | 1000亿-5000亿 | AI应用、数据中心、新能源车 |
| **成熟期** | >20% | 技术稳定迭代 | >5000亿 | 传统半导体、光伏、风电 |

---

## 行业阶段判断逻辑

```
IF 渗透率<5% AND (技术处于验证期 OR 市场规模<1000亿)
    → 行业阶段 = "起步期"
ELSE IF 渗透率5%-20% OR (技术规模化应用 AND 政策体系完善)
    → 行业阶段 = "成长期"
ELSE
    → 行业阶段 = "成熟期"
```

## 行业阶段与股票类型对应关系

| 行业阶段 | 可能的股票类型 | 识别条件 |
| :--- | :--- | :--- |
| **起步期** | 早期成长股 | 增速>10% + 领先地位 |
| **起步期** | 早期转型股 | 转向起步期行业 + 新业务占比>3% + 增速>30% |
| **成长期** | 高成长股 | 增速>50% |
| **成长期** | 成长股 | 增速20%-50% |
| **成长期** | 潜力成长股 | 增速0%-20% + 潜力指标支撑 |
| **成熟期** | 成长股 | 增速20%-50% |
| **成熟期** | 成熟股 | 增速<20% |
