> **重要**：以下约束在执行过程中必须严格遵守，违反将导致错误结果

| 约束项 | 要求 | 违反后果 | 检查时机 |
|--------|------|---------|---------|

## 步骤 5：股票类型识别（三层复合决策树）

**执行时机**：步骤4完成后，基于行业属性和财务数据判断

**操作内容**：

> **重要**：股票类型识别采用**三层复合体系**，必须严格按决策树执行。

### 三层识别流程

```
第一层：行业属性判断（步骤4.1已完成）
    ↓
第二层：财务指标验证（本步骤执行）
    ↓
第三层：前瞻指标微调（仅对潜力成长股）
```

### 完整决策树

```
【第一层：行业属性】（已在步骤4.1确定）
行业属性 = "高成长赛道" 或 "普通行业"

【第二层：财务指标验证】

【优先级1：亏损股】
IF 净利润（近3年平均）< 0
    → 最终类型 = 亏损股

【优先级2：周期股】
ELSE IF 近3年利润增速标准差 > 行业平均 × 2
    → 最终类型 = 周期股

【优先级3：转型股】
ELSE IF 满足以下任意2条：
    - 新业务营收占比 > 5% 且 新业务增速 > 50%
    - 新业务占比年度提升 > 3个百分点
    - 新业务所在行业CAGR > 15%
    - 研发占比 > 8% 且新方向投入 > 50%
    - 管理层明确宣布转型战略
    → 最终类型 = 转型股

【优先级4-7：成长类型判断（分行业属性）】

ELSE IF 行业属性 = "高成长赛道"
    IF 营收CAGR > 50%
        → 最终类型 = 高成长股
    ELSE IF 营收CAGR > 20%
        → 最终类型 = 成长股
    ELSE IF 营收CAGR > 0%
        → 候选"潜力成长股"（进入第三层）
    ELSE
        → 最终类型 = 成熟股

ELSE（普通行业）
    IF 营收CAGR > 50%
        → 最终类型 = 高成长股
    ELSE IF 营收CAGR > 15%
        → 最终类型 = 成长股
    ELSE
        → 最终类型 = 成熟股

【第三层：前瞻指标验证】（仅对候选"潜力成长股"）
IF 满足任一前瞻指标：
    - 在手订单/年收入 > 100%
    - 产能扩张计划 > 50%
    - 行业渗透率 < 20% 且公司为头部
    - 客户结构优化（新增头部客户）
    - 业务量增速 > 30%（机柜数/用户数/出货量等）
    → 最终类型 = 潜力成长股
ELSE
    → 最终类型 = 成熟股
```

### 股票类型汇总（7种类型）

| 股票类型 | 识别条件 | 特征描述 | 适用权重体系 |
| :--- | :--- | :--- | :--- |
| **高成长股** | 营收增速>50% | 业绩爆发期 | 高成长股权重 |
| **成长股** | 营收增速20%-50%（高成长赛道）或 15%-50%（普通行业） | 成长动能强劲 | 成长股权重 |
| **潜力成长股** | 高成长赛道 + 营收增速0%-20% + 前瞻指标支撑 | 产能爬坡期 | 潜力成长股权重 |
| **成熟股** | 增速<20%（高成长赛道）或 <15%（普通行业） | 稳定经营 | 成熟股权重 |
| **周期股** | 利润增速标准差 > 行业平均 × 2 | 盈利随周期波动 | 周期股权重 |
| **亏损股** | 净利润（近3年平均）< 0 | 投入期未盈利 | 亏损股权重 |
| **转型股** | 满足转型股条件（见上） | 向高成长赛道转型 | 转型股权重 |

### 识别依据记录

必须记录识别依据，包括：
- **第一层**：行业属性判断依据（如：数据中心 ∈ 高成长赛道）
- **第二层**：财务指标数据（如：营收CAGR 7%，净利润为正）
- **第三层**：前瞻指标数据（如：机柜数增速40%，在手订单2倍年收入）
- **最终结论**：股票类型 = 潜力成长股

### 识别示例

**案例：润泽科技（数据中心行业）**

```
【第一层：行业属性】
  数据中心 ∈ 高成长赛道清单 → 行业属性 = "高成长赛道" ✅

【第二层：财务指标】
  营收CAGR ≈ 7% < 20% → 候选"潜力成长股"

【第三层：前瞻指标】
  - 机柜数增速 40% > 30% ✅
  - 上架率 95%（接近满产）✅
  - 在手订单 2倍年收入 > 100% ✅
  → 满足前瞻指标 → 最终类型 = 潜力成长股 ✅
```

**输出结果**：
- ✅ 股票类型已确定（7种类型之一）
- ✅ 三层识别依据已完整记录

**7.2 报告内容要求**

除了 skill 自带内容外，**必须包含**以下章节：

**A. 股票类型识别**
- 明确说明股票属于哪种类型（6种之一）
- 提供识别依据（参考步骤5的决策树）

**B. 投资建议-综合评级**

必须包含以下评分内容：

**B.1 五个维度评分**（每项 1-5 分）

| 维度 | 评分依据 | 加扣分 | 最终评分 |
| :--- | :--- | :--- | :--- |
| 行业前景 | 引用评级定义文件中的量化标准 | 列出加分/扣分因素 | 1-5分 |
| 公司竞争力 | 引用评级定义文件中的量化标准 | 列出加分/扣分因素 | 1-5分 |
| 成长性 | 引用评级定义文件中的量化标准 | 列出加分/扣分因素 | 1-5分 |
| 财务健康度 | 引用评级定义文件中的量化标准 | 列出加分/扣分因素 | 1-5分 |
| 估值合理性 | 引用评级定义文件中的量化标准 | 列出加分/扣分因素 | 1-5分 |

**B.2 综合评分计算**

```
1. 说明使用的权重体系（根据步骤5识别的股票类型）
   示例："该股票为成长股，使用成长股权重：行业前景20%、公司竞争力25%、成长性30%、财务健康度15%、估值合理性10%"

2. 加权计算过程
   示例：
   基础综合评分 = 行业前景评分×权重 + 公司竞争力评分×权重 + ... = 3.65分

3. 成长系数判断（如适用）
   - 高成长股（增速>50%）：根据增速查表确定成长系数（1.00-1.15）
   - 转型股（新业务增速>50%）：根据新业务增速和占比确定成长系数（1.00-1.15）
   - 其他股票类型：成长系数=1.00（不调整）

4. 最终评分计算
   最终评分 = 基础综合评分 × 成长系数（上限5.0分）
   示例：3.65 × 1.10 = 4.02分
```

**B.3 综合评级**

- 根据评分范围对照评级定义给出对应评级
- 提供评级理由
- 若应用了成长系数（≠1.00），需说明成长系数的确定依据

**C. 投资逻辑**

核心投资逻辑的 3-5 点总结

**D. 投资论点总结**

看多/看空的关键论点归纳

3. **避免**：单元格内换行、过长内容、嵌套表格

**原因**：
- `评级星级` 不是根据 `评级分数` 范围确定
- 未使用正确的权重体系
- 股票类型识别错误

**解决**：
1. 重新识别股票类型（参考步骤5决策树）
2. 使用对应权重计算加权综合评分（参考评级定义文件）
3. 根据步骤8.2的评分范围对照表确定评级星级
4. 评级由综合评分决定，必须严格遵循评级定义文件

**预防**：严格按照步骤5-8的顺序执行，不要跳过股票类型识别步骤

---

### 问题 5：评级图标显示"未评级"

**原因**：缺少 `评级星级` 字段或值不在 1-5 范围内

**解决**：
1. 确保 YAML 中包含 `评级星级` 字段
2. 确认 `评级星级` 值为 1-5 的整数

**预防**：使用步骤8提供的标准化 YAML 模板

---

### 问题 6：必需数据缺失过多

**原因**：P1-P3必需数据缺失超过30%

**解决**：
1. 中止当前分析
2. 向用户报告缺失的数据项
3. 等待用户补充数据或确认继续

**预防**：在步骤4严格执行数据查询，遇到缺失立即标注

---

### 问题 7：多股票处理时上下文混乱

**原因**：
- 并行处理多只股票
- 未执行步骤11清理上下文
- 清理上下文后未重新执行步骤3

**解决**：
1. 确保每次只处理一只股票
2. 每只股票完成后执行 `/clear`
3. 清理后返回步骤3重新加载配置文件

**预防**：严格遵守步骤11的清理流程和步骤12的循环控制

---

### 问题 8：字段名使用错误（如"是否更新"而非"是否有更新"）

**原因**：未仔细阅读 base 文件的字段定义

**解决**：
1. 重新读取 base 文件（步骤3.2）
2. 确认字段名与 base 公式引用完全一致
3. 修正 YAML 字段名

**预防**：使用步骤8提供的标准化 YAML 模板，或使用步骤0的字段名对照表

### 股票类型识别（7种，三层复合体系）

> **详细定义请参考**：`分析脚本/投资评级定义.md` → "股票类型识别（三层复合体系）"章节

| 类型 | 识别标准 | 优先级 | 适用场景 |
|------|---------|-------|---------|
| 亏损股 | 净利润（近3年平均）< 0 | 1（最高） | 投入期未盈利公司 |
| 周期股 | 近3年利润增速标准差 > 行业平均 × 2 | 2 | 盈利波动大 |
| 转型股 | 新业务占比>5% 且 新业务增速>50%（满足2条） | 3 | 向高成长赛道转型 |
| 高成长股 | 营收CAGR > 50% | 4 | 业绩爆发期 |
| 成长股 | 营收CAGR 20%-50%（高成长赛道）或 15%-50%（普通行业） | 5 | 成长动能强劲 |
| 潜力成长股 | 高成长赛道 + 营收CAGR 0%-20% + 前瞻指标支撑 | 6 | 产能爬坡期 |
| 成熟股 | 其他 | 7（最低） | 稳定经营 |

**三层识别要点**：
1. **第一层**：行业属性（高成长赛道/普通行业）
2. **第二层**：财务指标验证（增速、利润稳定性）
3. **第三层**：前瞻指标微调（仅对潜力成长股）

### 评级等级（5级）

| 评级 | 综合评分范围 | 评级星级 | 预期收益特征 |
|------|-------------|---------|-------------|
| 强力买入 | 4.5 - 5.0 | 5 | 显著跑赢大盘，超基准 15%-20%+ |
| 买入 | 4.0 - 4.4 | 5 | 涨幅高于基准 10%-20% |
| 增持 | 3.5 - 3.9 | 4 | 小幅跑赢或优于大盘 5%-15% |
| 观望 | 2.5 - 3.4 | 3 | 与大盘基本持平（±5% 以内） |
| 回避 | 1.0 - 2.4 | 1 | 跑输大盘或下跌超 5%-15% |

### 成长系数机制（仅高成长股和转型股适用）

| 增速范围 | 成长系数 | 适用股票类型 |
|---------|---------|-------------|
| >100% | 1.15 | 高成长股、转型股 |
| 80%-100% | 1.10 | 高成长股、转型股 |
| 60%-80% | 1.05 | 高成长股、转型股 |
| 50%-60% | 1.00 | 高成长股、转型股 |
| <50% | 1.00 | 所有股票类型 |

**最终评分计算公式**：
```
最终评分 = 基础综合评分 × 成长系数（上限5.0分）
```
