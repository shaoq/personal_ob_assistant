we# 投资评级定义

本评级体系基于五维度评分系统（行业前景、公司竞争力、成长性、财务健康度、估值合理性），综合评估后给出投资建议。

## 评级等级与定义

| 评级       | 预期收益   | 基准比较                | 风险程度 | 适合投资者  |
| -------- | ------ | ------------------- | ---- | ------ |
| **强力买入** | 显著跑赢大盘 | 超基准指数 15%-20%+      | 中高风险 | 激进型投资者 |
| **买入**   | 看好     | 涨幅高于基准 10%-20%      | 中等风险 | 积极型投资者 |
| **增持**   | 谨慎推荐   | 小幅跑赢或优于大盘 5%-15%    | 中低风险 | 稳健型投资者 |
| **观望**   | 中性判断   | 与大盘基本持平（±5% 以内）     | 低风险  | 保守型投资者 |
| **回避**   | 看空     | 跑输大盘或下跌（跌幅超 5%-15%） | 高风险  | 不建议投资  |

## 评级与五维度评分对应关系

| 评级 | 综合评分范围 | 评分特征 |
| :--- | :----------- | :------- |
| **强力买入** | 4.5 - 5.0 分 | 五维度均优秀，行业景气度极高，公司竞争力突出，成长性强劲，财务健康，估值合理；高成长股/转型股可能通过成长系数（1.05-1.15）提升评分 |
| **买入** | 4.0 - 4.4 分 | 多数维度优秀，行业景气度高，公司竞争力强，成长性好，财务状况良好，估值相对合理 |
| **增持** | 3.5 - 3.9 分 | 整体表现良好，有一定亮点，但存在部分短板（如估值偏高或财务一般） |
| **观望** | 2.5 - 3.4 分 | 优缺点并存，行业或公司存在不确定性，建议等待更好入场时机或观望 |
| **回避** | 1.0 - 2.4 分 | 行业景气度差，公司竞争力弱，财务状况不佳，估值过高或存在重大风险 |

### 行业识别方法

1. **精确匹配**：公司主营业务描述与高成长赛道子行业名称匹配
2. **关键字匹配**：主营业务描述包含高成长赛道识别关键字
3. **关联匹配**：公司主要客户/供应商属于高成长赛道

## 股票类型识别（三层复合体系）

> **核心思路**：从单一财务指标识别 → 多维复合识别（行业属性 + 财务指标 + 前瞻指标）

### 三层识别流程

```
第一层：行业属性判断（决定基准分类）
    ↓
第二层：财务指标验证（修正分类）
    ↓
第三层：前瞻指标微调（可选，最终确认）
```

### 第一层：行业属性判断

```
IF 公司所属行业 ∈ 高成长赛道清单
    → 行业属性 = "高成长赛道"
ELSE
    → 行业属性 = "普通行业"
```

### 第二层：财务指标验证（决策树）

```
【优先级1：亏损股】
IF 净利润（近3年平均）< 0
    → 最终类型 = 亏损股

【优先级2：周期股】
ELSE IF 利润增速标准差 > 行业平均 × 2
    → 最终类型 = 周期股

【优先级3：转型股】
ELSE IF 满足转型股条件（见下表）
    → 最终类型 = 转型股

【优先级4-6：成长类型判断】
ELSE IF 行业属性 = "高成长赛道"
    IF 营收CAGR > 50%
        → 最终类型 = 高成长股
    ELSE IF 营收CAGR > 20%
        → 最终类型 = 成长股
    ELSE IF 营收CAGR > 0%
        → 候选"潜力成长股"（进入第三层）
    ELSE
        → 最终类型 = 成熟股

ELSE（普通行业）
    IF 营收CAGR > 50%
        → 最终类型 = 高成长股
    ELSE IF 营收CAGR > 15%
        → 最终类型 = 成长股
    ELSE
        → 最终类型 = 成熟股
```

### 第三层：前瞻指标验证（仅对"潜力成长股"）

**触发条件**：高成长赛道 + 营收CAGR 0%-20%

**前瞻指标清单**（满足任一即可）：

| 前瞻指标 | 判断标准 | 数据来源 |
| :--- | :--- | :--- |
| 在手订单/年收入 | >100% | 公司公告、年报 |
| 产能扩张计划 | >50% | 公司公告、资本开支计划 |
| 行业渗透率 | <20% 且 公司为头部 | 行业研究报告 |
| 客户结构优化 | 新增头部客户 | 公司公告、调研纪要 |
| 业务量增速 | >30%（机柜数/用户数/出货量等） | 公司公告、年报 |

**判断逻辑**：
```
IF 满足任一前瞻指标
    → 最终类型 = 潜力成长股
ELSE
    → 最终类型 = 成熟股
```

### 股票类型汇总表

| 股票类型 | 识别条件 | 特征描述 |
| :--- | :--- | :--- |
| **高成长股** | 营收增速>50% | 处于业绩爆发期，高速扩张 |
| **成长股** | 营收增速20%-50%（高成长赛道）或 15%-50%（普通行业） | 成长动能强劲 |
| **潜力成长股** | 高成长赛道 + 营收增速0%-20% + 前瞻指标支撑 | 处于产能爬坡期，未来增长可期 |
| **成熟股** | 增速<20%（高成长赛道）或 <15%（普通行业） | 稳定经营，增长缓慢 |
| **周期股** | 近3年利润增速标准差 > 行业平均 × 2 | 盈利随周期波动 |
| **亏损股** | 净利润（近3年平均）< 0 | 处于投入期，尚未盈利 |
| **转型股** | 满足以下任意2条：<br>1. 新业务营收占比>5% 且增速>50%<br>2. 新业务占比年度提升>3个百分点<br>3. 新业务所在行业CAGR>15%<br>4. 研发占比>8% 且新方向投入>50%<br>5. 管理层明确宣布转型战略 | 从传统业务向高成长赛道转型 |

> **识别优先级**：亏损股 → 周期股 → 转型股 → 高成长股 → 成长股 → 潜力成长股 → 成熟股

### 三层识别示例

**案例：润泽科技（数据中心行业）**

```
【第一层：行业属性】
  数据中心 ∈ 高成长赛道清单 → 行业属性 = "高成长赛道" ✅

【第二层：财务指标】
  营收CAGR ≈ 7% < 20% → 候选"潜力成长股"

【第三层：前瞻指标】
  - 机柜数增速 40% > 30% ✅
  - 上架率 95%（接近满产）✅
  - 在手订单 2倍年收入 > 100% ✅
  → 满足前瞻指标 → 最终类型 = 潜力成长股 ✅
```

## 五维度评分标准

**⚠️ 重要**：最终评分 = 基础评分 + 所有加分 - 所有扣分，最低不低于1分，最高不超过5分

> **计算步骤**：
> 1. **先加总所有加分**： `总加分 = 加分1 + 加分2 + ...`
> 2. **再加总所有扣分**： `总扣分 = 扣分1 + 扣分2 + ...`
> 3. **最后判断是否超出范围**：
>    - 若总分 > 5分，取 **5分**
>    - 若总分 < 1分，取 **1分**

### 1. 行业前景（1-5分）

**量化评分标准**：

| 分数 | 市场规模CAGR | 政策支持度 | 技术变革 | 竞争格局 | 典型行业 |
| :--- | :----------- | :--------- | :------- | :------- | :------- |
| 5分 | >20% | 强力支持（国家级战略） | 颠覆性技术突破 | 蓝海市场 | AI、人形机器人、量子计算 |
| 4分 | 10%-20% | 明确支持（产业政策） | 技术快速迭代 | 格局初定 | 新能源、半导体、创新药 |
| 3分 | 5%-10% | 中性或一般支持 | 技术稳定升级 | 格局稳定 | 消费电子、家电、汽车 |
| 2分 | 0%-5% | 政策收紧或无支持 | 技术停滞 | 竞争加剧 | 传统制造、地产链 |
| 1分 | 负增长 | 政策限制/淘汰 | 技术被替代 | 恶性竞争 | 煤炭、钢铁、水泥 |

**扣分因素**：

| 风险因素 | 扣分 |
| :--- | :--- |
| 行业处于周期高点，产能过剩风险 | -0.5分 |
| 存在重大技术替代风险 | -1.0分 |
| 政策存在不确定性（如集采、补贴退坡） | -0.5分 |

**加分因素**：

| 加分因素 | 加分 |
| :--- | :--- |
| 行业渗透率<10%，处于早期爆发阶段 | +0.5分 |
| 全球竞争力强，出海空间大 | +0.5分 |

### 2. 公司竞争力（1-5分）

**量化评分标准**：

| 分数 | 市场地位 | 技术壁垒 | 客户资源 | 毛利率 | 典型特征 |
| :--- | :------- | :------- | :------- | :----- | :------- |
| 5分 | 全球/国内绝对龙头（份额>30%） | 极高（专利壁垒/独家技术） | 顶级客户（全球头部企业） | >50% | 拥有定价权，难以被替代 |
| 4分 | 国内领先（份额15%-30%） | 高（核心专利/技术积累） | 优质客户（行业头部） | 35%-50% | 品牌认知度高，技术领先 |
| 3分 | 行业中游（份额5%-15%） | 中等（有一定技术积累） | 客户稳定但分散 | 25%-35% | 产品同质化，价格竞争 |
| 2分 | 行业下游（份额<5%） | 较低（技术门槛不高） | 客户质量一般 | 15%-25% | 竞争激烈，议价能力弱 |
| 1分 | 无明显地位 | 无壁垒 | 客户不稳定 | <15% | 处于淘汰边缘 |

**扣分因素**：

| 风险因素 | 扣分 |
| :--- | :--- |
| 核心技术依赖外部授权/进口 | -0.5分 |
| 主要客户流失风险（>20%收入来源不稳） | -1.0分 |
| 产品被替代风险高 | -0.5分 |

**加分因素**：

| 加分因素 | 加分 |
| :--- | :--- |
| 一体化程度高（上下游延伸） | +0.5分 |
| 研发投入占比>10% | +0.5分 |
| 拥有独家/垄断性资源 | +1.0分 |
| 海外收入占比>30%（全球化布局） | +0.5分 |

### 3. 成长性（1-5分）

**量化评分标准**：

| 分数 | 营收增速 | 利润增速 | 订单/在手订单 | 行业增速对比 | 典型特征 |
| :--- | :------- | :------- | :------------ | :----------- | :------- |
| 5分 | >50% | >50% | 爆发式增长（同比>100%） | 显著超越行业 | 处于业绩爆发期 |
| 4分 | 30%-50% | 30%-50% | 高速增长（同比50%-100%） | 领先行业 | 成长动能强劲 |
| 3分 | 10%-30% | 10%-30% | 稳定增长（同比20%-50%） | 与行业持平 | 稳健成长 |
| 2分 | 0%-10% | 0%-10% | 增长疲软或下滑 | 落后行业 | 增长停滞 |
| 1分 | 负增长 | 负增长 | 萎缩 | 显著落后行业 | 业绩下滑 |

**扣分因素**：

| 风险因素 | 扣分 |
| :--- | :--- |
| 增速连续2个季度下滑（趋势恶化） | -0.5分 |
| 增长依赖并购/非经常性损益 | -0.5分 |
| 增长不增收（增收不增利） | -0.5分 |

**加分因素**：

| 加分因素 | 加分 |
| :--- | :--- |
| 在手订单覆盖年收入>1.5倍 | +0.5分 |
| 新业务/新产品占比快速提升 | +0.5分 |
| 连续3年增速>30%（业绩确定性高） | +0.5分 |

> **注意**：利润增速优先于营收增速，若利润增速显著高于营收增速（规模效应），可适当加分

### 4. 财务健康度（1-5分）

**基础评分标准**（基于财务指标）：

| 分数 | 资产负债率 | 现金流 | 应收账款 | ROE | 净利率 |
| :--- | :--------- | :----- | :------- | :-- | :----- |
| 5分 | <30% | 充裕（经营现金流/净利润>1.2） | 健康（周转天数<60天） | >20% | >20% |
| 4分 | 30%-50% | 良好（经营现金流/净利润>1.0） | 可控（周转天数60-90天） | 15%-20% | 15%-20% |
| 3分 | 50%-60% | 一般（经营现金流/净利润>0.8） | 偏高（周转天数90-120天） | 10%-15% | 10%-15% |
| 2分 | 60%-70% | 紧张（经营现金流/净利润<0.8） | 高企（周转天数120-180天） | 5%-10% | 5%-10% |
| 1分 | >70% | 为负或持续恶化 | 严重（周转天数>180天） | <5% | <5% |

**扣分因素**（经营风险评估）：

| 风险因素 | 扣分 |
| :--- | :--- |
| 客户集中度 >50%（前五大客户占比） | -0.5分 |
| 客户集中度 >70% | -1.0分 |
| 客户集中度 >80% | -1.5分 |
| 供应商集中度 >60% | -0.5分 |
| 存货周转率显著下降（同比>20%） | -0.5分 |
| 存在重大诉讼或监管处罚 | -1.0分 |

**加分因素**：

| 加分因素 | 加分 |
| :--- | :--- |
| 现金储备充足（现金/总资产>30%） | +0.5分 |
| 连续3年分红且股息率>2% | +0.5分 |
| 无有息负债 | +0.5分 |

> **注意**：最终评分 = 基础评分 + 加分 - 扣分，最低不低于1分，最高不超过5分

### 5. 估值合理性（1-5分）

**估值评估原则**：
- 优先使用**动态PE**（基于未来12个月预期EPS计算），更能反映成长股的真实估值
- 对于高成长公司（增速>30%），重点参考**PEG指标**
- 对于成熟公司（增速<15%），重点参考**PE/PB相对行业均值的位置**

**基础评分标准**（优先使用动态估值）：

| 分数 | 动态PE相对行业 | PEG | PB相对行业 | 说明 |
| :--- | :------------ | :-- | :--------- | :--- |
| 5分 | 低于行业均值20%以上 | <1.0 | 低于行业均值 | 安全边际充足，显著低估 |
| 4分 | 接近或略低于行业均值（±10%） | 1.0-1.5 | 接近行业均值 | 估值合理 |
| 3分 | 高于行业均值10%-30% | 1.5-2.0 | 高于行业均值10%-30% | 估值偏高但可接受（需成长性支撑） |
| 2分 | 高于行业均值30%-50% | 2.0-3.0 | 高于行业均值30%-50% | 估值泡沫明显 |
| 1分 | 高于行业均值50%以上 | >3.0 | 高于行业均值50%以上 | 严重高估 |

**调整因素**：

| 情况 | 调整 |
| :--- | :--- |
| 业绩增速>50%，可适当放宽PE要求 | +0.5分（但PEG仍需<2） |
| 处于行业爆发期，市场给予估值溢价 | +0.5分（最高不超过4分） |
| 近期有重大利空导致股价暴跌 | -0.5分（可能存在价值陷阱） |
| 股价显著高于相对估值区间（DCF/NAV等） | -0.5分 |

**静态PE与动态PE的使用场景**：

| 场景 | 使用指标 | 原因 |
| :--- | :------- | :------- |
| 成长股（增速>30%） | 动态PE + PEG | 业绩快速增长可消化高估值 |
| 周期股 | 静态PE（取周期均值） | 盈利波动大，单年PE失真 |
| 成熟股（增速<15%） | 静态PE + PB | 增长稳定，历史数据更可靠 |
| 亏损或微利股 | PB + PS | PE失效，用其他指标替代 |

> **注意**：对于高成长公司，即使静态PE较高（如100倍以上），若动态PE合理（如30倍以下）且PEG<1.5，仍可给予3-4分

## 动态权重评分体系

### 权重分配原则

不同类型股票的核心投资逻辑不同，因此各维度权重应动态调整：

| 股票类型 | 行业前景 | 公司竞争力 | 成长性 | 财务健康度 | 估值合理性 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **高成长股**（增速>50%） | 25% | 20% | 35% | 10% | 10% |
| **成长股**（增速20%-50%） | 22% | 22% | 30% | 13% | 13% |
| **潜力成长股**（高成长赛道+前瞻指标支撑） | 25% | 23% | 20% | 18% | 14% |
| **成熟股**（增速<20%） | 18% | 22% | 10% | 28% | 22% |
| **周期股** | 25% | 20% | 12% | 23% | 20% |
| **亏损股**（营收增长中） | 28% | 25% | 30% | 10% | 7% |
| **转型股** | 28% | 22% | 25% | 15% | 10% |

### 权重设计逻辑

#### 高成长股（增速>50%）
- **成长性35%**：核心优势维度，给予最高权重
- **行业前景25%**：成长股高度依赖行业景气度
- **公司竞争力20%**：执行力和技术壁垒重要
- **财务健康度10%**：扩张期财务指标天然偏低，降低权重
- **估值合理性10%**：静态PE失真，降低权重

#### 潜力成长股（高成长赛道+前瞻指标支撑）
- **行业前景25%**：身处高成长赛道，赋予较高权重
- **公司竞争力23%**：验证公司是否能抓住行业机会
- **成长性20%**：当前增速不高，但前瞻指标支撑未来增长
- **财务健康度18%**：适度关注财务稳健性
- **估值合理性14%**：成长期公司估值波动大，适度降低

#### 成熟股（增速<15%）
- **财务健康度28%**：成熟公司财务稳健性是核心
- **估值合理性22%**：静态估值指标更可靠
- **成长性10%**：增长已不是核心驱动力

#### 周期股
- **行业前景25%**：周期位置决定盈利
- **财务健康度23%**：穿越周期的能力关键

#### 亏损股（营收增长中）
- **行业前景28%**：亏损股高度依赖行业赛道和景气度
- **公司竞争力25%**：技术和市场地位决定能否盈利
- **成长性30%**：营收增速是核心观察指标
- **财务健康度10%**：现金流和现金储备比利润更重要
- **估值合理性7%**：PE失效，主要参考PS/PB，权重最低

### 转型股评分方法

转型股是从传统业务向高成长赛道转型的企业，需要特殊的评分方法来准确反映其投资价值。

#### 行业前景"双轨制"评分

转型股需要同时对**原主业**和**新业务**分别评分，然后加权：

```
行业前景综合评分 = 原主业评分 × 原主业权重 + 新业务评分 × 新业务权重
```

**权重确定原则**：

| 转型阶段 | 原主业权重 | 新业务权重 | 判断依据 |
| :--- | :--- | :--- | :--- |
| 早期（新业务占比<10%） | 70% | 30% | 新业务贡献小但潜力大 |
| 中期（新业务占比10%-30%） | 50% | 50% | 双轮驱动 |
| 后期（新业务占比>30%） | 30% | 70% | 新业务已成为核心 |

**示例**：
- 汽车制造企业切入机器人赛道
- 原主业（汽车）：3分（成熟行业）
- 新业务（机器人）：5分（高景气赛道）
- 新业务占比15%（中期转型）
- 行业前景综合 = 3×50% + 5×50% = 4分

#### 成长性评分（加入前瞻性指标）

转型股的成长性评分应结合**当前增速**和**未来潜力**：

| 分数 | 当前增速 | 前瞻指标（满足任一） |
| :--- | :------- | :--- |
| 5分 | >30% 或 新业务增速>100% | 新业务订单爆发（>100%）、产能已就绪 |
| 4分 | 15%-30% 或 新业务增速>50% | 新业务订单高增长、客户拓展顺利 |
| 3分 | 5%-15% 或 新业务增速>30% | 新业务稳步推进、有明确里程碑 |
| 2分 | 0%-5% | 新业务进展缓慢 |
| 1分 | 负增长 | 转型失败或停滞 |

**加分因素**（转型股专属）：

| 加分因素 | 加分 |
| :--- | :--- |
| 新业务在手订单覆盖预期产能>80% | +0.5分 |
| 新业务已获得头部客户验证 | +0.5分 |
| 新业务毛利率高于原主业 | +0.5分 |
| 管理层有成功转型历史 | +0.5分 |

**扣分因素**（转型股专属）：

| 扣分因素 | 扣分 |
| :--- | :--- |
| 转型已持续3年以上但新业务仍未盈利 | -0.5分 |
| 新业务资本开支过大导致现金流紧张 | -1.0分 |
| 原主业快速恶化拖累转型资源 | -0.5分 |

#### 转型股权重设计逻辑

| 维度 | 权重 | 设计逻辑 |
| :--- | :--- | :--- |
| 行业前景 | 28% | 双赛道前景，权重提高 |
| 公司竞争力 | 22% | 执行力是转型成功关键 |
| 成长性 | 25% | 兼顾当前增速和未来潜力 |
| 财务健康度 | 15% | 转型期财务压力天然偏大，适度降低 |
| 估值合理性 | 10% | 转型期估值波动大，降低权重 |

### 亏损股估值方法

对于尚未盈利的公司，估值评估需要特殊处理：

| 方法 | 说明 | 适用场景 |
| :--- | :--- | :--- |
| **PS（市销率）** | 市值/营收，替代PE | 营收稳定增长但未盈利 |
| **PB（市净率）** | 市值/净资产 | 资产较重的公司 |
| **现金消耗率** | 月均烧钱速度 | 评估生存能力 |
| **现金跑道** | 账面现金可支撑月数 | 评估融资紧迫性 |

## 成长系数机制

### 设计目的

在现有动态权重评分体系基础上，通过**成长系数**来体现超高成长性的额外价值：

```
最终综合评分 = 基础综合评分 × 成长系数
```

**优势**：
1. 不破坏现有的五维度评分和动态权重体系
2. 灵活体现超高成长性的额外价值
3. 仅对符合条件的股票生效，不影响其他股票

### 适用范围

成长系数**仅适用于以下情况**：
- **高成长股**（整体增速>50%）
- **转型股**（新业务增速>50%）

其他股票类型（成熟股、周期股、亏损股）**不适用**成长系数。

### 成长系数档位表

**高成长股成长系数**

| 整体增速范围 | 成长系数 | 说明 |
| :--- | :--- | :--- |
| 50%-80% | 1.00 | 基础高成长，不调整 |
| 80%-120% | 1.05 | 超高成长 |
| 120%-200% | 1.10 | 爆发式成长 |
| >200% | 1.15 | 极端爆发（罕见） |

**转型股成长系数**

| 新业务增速 | 新业务占比 | 成长系数 | 说明 |
| :--- | :--- | :--- | :--- |
| 50%-100% | 任意 | 1.00 | 基础高成长转型 |
| 100%-150% | >10% | 1.05 | 高成长转型 |
| 150%-200% | >10% | 1.10 | 超高成长转型 |
| **>200%** | **>15%** | **1.15** | **爆发式转型** |
| >200% | 10%-15% | 1.10 | 占比略低，适度调整 |
| >100% | <10% | 1.00 | 占比过低，不调整 |

### 成长系数上限约束

**重要约束**：
- 成长系数最高为 **1.15**
- 最终综合评分**不得超过5.0分**（评级体系上限）
- 若基础评分 × 成长系数 > 5.0，则取 **5.0分**

---

## 成长系数机制使用说明

### 评分计算流程

1. **收集数据**：获取公司财务数据、行业信息、市场数据
2. **分维度评分**：
   - 根据量化标准确定基础评分
   - 应用加分/扣分因素
   - 确保最终评分在1-5分范围内
3. **识别股票类型**：根据增速指标判断股票类型（高成长股/成长股/成熟股/周期股/亏损股/转型股）
4. **计算基础综合评分**：
   - 根据股票类型选择对应权重
   - 计算加权综合评分
5. **判断是否适用成长系数**：
   - 仅高成长股（增速>50%）和转型股（新业务增速>50%）适用
   - 其他股票类型跳过此步骤
6. **确定成长系数**：
   - 根据成长系数档位表确定系数（1.00-1.15）
   - 若不适用则系数为1.00
7. **计算最终综合评分**：
   - 最终评分 = 基础综合评分 × 成长系数
   - 上限5.0分（若超过则取5.0）
8. **确定评级**：根据最终综合评分范围确定最终评级

## 评分计算示例

### 案例A：高成长半导体设备公司（增速>50%）

**五维度评分**：

| 维度 | 基础评分 | 加分因素 | 扣分因素 | 最终评分 |
| :--- | :------- | :------- | :------- | :------- |
| 行业前景 | 5（CAGR>20%，政策强力支持） | +0.5（渗透率低） | - | 5.0 |
| 公司竞争力 | 4（国内领先，技术壁垒高） | +0.5（研发>10%） | - | 4.5 |
| 成长性 | 5（营收增速60%，订单爆发） | +0.5（在手订单充足） | - | 5.0 |
| 财务健康度 | 5（负债率15%，ROE 35%） | +0.5（现金充裕） | -1.0（客户集中度>70%） | 4.5 |
| 估值合理性 | 3（动态PE高于行业30%） | +0.5（业绩增速快） | - | 3.5 |

**方法1：等权重平均**（传统方式）
```
综合评分 = (5.0 + 4.5 + 5.0 + 4.5 + 3.5) / 5 = 4.5分 → 强力买入
```

**方法2：动态权重 + 成长系数**（优化方式）
```
步骤1：股票类型识别
  → 高成长股（增速60%>50%）

步骤2：计算基础综合评分
  加权评分 = 5.0×25% + 4.5×20% + 5.0×35% + 4.5×10% + 3.5×10%
           = 1.25 + 0.90 + 1.75 + 0.45 + 0.35
           = 4.70分

步骤3：判断成长系数
  增速60%，处于50%-80%档位 → 成长系数 = 1.00（基础档位，不调整）

步骤4：计算最终评分
  最终评分 = 4.70 × 1.00 = 4.70分 → 强力买入
```

**结论**：增速60%处于基础档位（50%-80%），成长系数为1.00，评分保持4.70分 → 强力买入。动态权重已充分体现高成长股特点。

### 案例B：财务健康度偏低的成长股（增速>50%）

**五维度评分**：

| 维度 | 最终评分 | 说明 |
| :--- | :------- | :------- |
| 行业前景 | 4.5 | 高景气赛道 |
| 公司竞争力 | 4.0 | 技术领先 |
| 成长性 | 5.0 | 业绩爆发期 |
| 财务健康度 | 2.5 | 高速扩张，负债率高 |
| 估值合理性 | 2.5 | 静态PE高 |

**方法1：等权重平均**（传统方式）
```
综合评分 = (4.5 + 4.0 + 5.0 + 2.5 + 2.5) / 5 = 3.7分 → 增持
```

**方法2：动态权重**（优化方式）
```
股票类型识别：高成长股（增速>50%）
加权评分 = 4.5×25% + 4.0×20% + 5.0×35% + 2.5×10% + 2.5×10%
         = 1.125 + 0.80 + 1.75 + 0.25 + 0.25
         = 4.175分 → 买入
```

**结论**：
- **等权重平均**：3.7分 → 增持（财务/估值拖累过重）
- **动态权重**：4.175分 → 买入（成长性权重35%充分体现核心优势）
- 动态权重已通过提升成长性权重至35%来体现高成长股特点，评分更合理

### 案例C：成熟股（增速<15%）

**五维度评分**：

| 维度 | 最终评分 | 说明 |
| :--- | :------- | :------- |
| 行业前景 | 3.0 | 行业稳定 |
| 公司竞争力 | 4.0 | 市场地位稳固 |
| 成长性 | 2.5 | 增速放缓 |
| 财务健康度 | 4.5 | 财务稳健 |
| 估值合理性 | 4.0 | 估值合理 |

**计算**：
```
股票类型识别：成熟股（增速<15%）
加权评分 = 3.0×18% + 4.0×22% + 2.5×10% + 4.5×28% + 4.0×22%
         = 0.54 + 0.88 + 0.25 + 1.26 + 0.88
         = 3.81分 → 增持
```

**对比等权重平均**：
```
综合评分 = (3.0 + 4.0 + 2.5 + 4.5 + 4.0) / 5 = 3.6分 → 增持
```

**结论**：成熟股评分与等权重平均接近，保持了向后兼容性。

### 案例D：转型股（汽车制造企业切入机器人赛道）

**基本信息**：
- 原主业：汽车零部件，占比70%，增速5%
- 新业务：机器人执行器，占比15%，增速200%
- 研发投入占比10%，其中70%投向机器人

**股票类型识别**：
- ✅ 新业务营收占比15%>5% 且增速200%>50%
- ✅ 新业务占比同比提升8个百分点>3%
- ✅ 机器人行业CAGR>20%>15%
- ✅ 研发投入10%>8% 且新方向70%>50%
- → **识别为转型股**（满足4条，超过2条阈值）

**五维度评分**：

| 维度 | 评分 | 计算说明 |
| :--- | :--- | :--- |
| 行业前景 | 4.4 | 汽车行业3分×70% + 机器人行业5分×30% = 3.6，+0.5（机器人渗透率低），+0.3（出海空间）= 4.4 |
| 公司竞争力 | 4.0 | 汽车领域龙头4分，机器人领域技术积累中等，综合4分 |
| 成长性 | 4.5 | 当前增速一般，但新业务增速200%，+0.5（新业务订单爆发），= 4.5 |
| 财务健康度 | 3.5 | 转型投入导致负债率偏高，但有现金支撑 |
| 估值合理性 | 3.0 | 市场给予转型溢价，PE高于行业均值 |

**综合评分计算（动态权重 + 成长系数）**：
```
步骤1：计算基础综合评分
  股票类型识别：转型股（新业务占比15%，中期转型阶段）
  加权评分 = 4.4×28% + 4.0×22% + 4.5×25% + 3.5×15% + 3.0×10%
           = 1.232 + 0.88 + 1.125 + 0.525 + 0.30
           = 4.062分

步骤2：判断是否适用成长系数
  转型股，新业务增速200%>50% → 适用成长系数

步骤3：确定成长系数
  新业务增速200%（>200%档位），新业务占比15%（>15%）
  → 查转型股成长系数表：成长系数 = 1.15（爆发式转型）

步骤4：计算最终评分
  最终评分 = 4.062 × 1.15 = 4.67分 → 强力买入
```

**效果对比**：

| 方法 | 最终评分 | 评级 | 说明 |
| :--- | :--- | :--- | :--- |
| 传统成熟股方法 | 3.56分 | 增持 | 低估转型价值 |
| 转型股动态权重（无成长系数） | 4.06分 | 买入 | 体现转型潜力 |
| 转型股动态权重 + 成长系数 | **4.67分** | **强力买入** | **充分体现新业务爆发式成长** |

**结论**：
- 转型股通过成长系数机制（1.15），评分从4.06分提升至4.67分
- 评级从"买入"提升到"强力买入"，准确反映新业务爆发式成长（200%增速）的投资价值
- 成长系数机制有效解决了转型股被低估的问题

## 重要提醒

1. **一致性原则**：同一公司在不同时期评分时，应使用相同的评分标准
2. **数据时效性**：使用最新的财务数据和市场信息
3. **主观判断最小化**：严格按照量化标准评分，减少主观因素
4. **特殊情况说明**：如有偏离标准的评分，需在报告中说明原因
5. **定期回顾**：建议每季度或重大事件后重新评估评级

> 根据综合评级定义创建对应的投资评级定义文件，确保投资分析报告的综合评级在上述定义范围内。

## 精细评分使用方式（Base 文件特有）

除了使用综合评级映射的整数评分，还可以使用 `评分` 字段提供更精细的评分（如 3.5、4.2）。在报告的 YAML frontmatter 中添加 `评分` 字段：

```yaml
---
股票名称: 东田微
股票代码: 301183
综合评级: 增持
评分: 3.5
---
```

**评分显示逻辑**：
- 有精细评分时：显示原始数值（如 `3.5`、`4.2`）
- 无精细评分时：根据综合评级显示对应整数分数（`5`、`4`、`3`、`2`、`1`）

**评分优先级**：如果同时存在 `评分` 和 `综合评级`，评分数值将优先使用 `评分` 字段的值。
