## 步骤 5：股票类型识别（三层复合决策树）

> **重要**：股票类型识别采用**三层复合体系**，必须严格按决策树执行。

### 前置检查：数据就绪确认

| 检查项 | 数据来源 |
| :--- | :--- |
| 行业属性已确定 | P2/P2.1 |
| 净利润近3年数据 | P1/P2.5 |
| 营收CAGR | P1 |
| 利润增速标准差 | P2.5 |
| 新业务相关数据（如适用） | P2.5 |
| 前瞻指标数据（如适用） | P2.5 |

### 三层识别流程

```
第一层：行业属性判断 + 行业阶段判断（步骤4.1已完成）
    ↓
第二层：财务指标验证（本步骤执行）
    ↓
第三层：前瞻指标验证（仅对潜力成长股）
```

### 完整决策树

```
【第一层：行业属性 + 行业阶段】
行业属性 = "高成长赛道" 或 "普通行业"
行业阶段 = "起步期" 或 "成长期" 或 "成熟期" 或 null

【第二层：财务指标验证】

【优先级1：亏损股】
IF 净利润（近3年平均）< 0
    → 最终类型 = 亏损股

【优先级2：周期股】
ELSE IF 近3年利润增速标准差 > 行业平均 × 2
    → 最终类型 = 周期股

【优先级3：转型股/早期转型股】
ELSE IF 满足转型股条件
    IF 新业务所处行业阶段 = "起步期"
        → 最终类型 = 早期转型股
    ELSE
        → 最终类型 = 转型股

【优先级3.5：潜力转型股】
ELSE IF 满足以下2条中任意1条：
    - 新业务营收占比 3%-10% 且 新业务增速 > 100%
    - 新业务营收占比 > 3% 且 新业务所在行业CAGR > 15%
    → 最终类型 = 潜力转型股

【优先级4：早期成长股】
ELSE IF 行业属性 = "高成长赛道" AND 行业阶段 = "起步期"
    IF 营收CAGR > 10% AND 公司地位领先
        → 最终类型 = 早期成长股
    ELSE IF 营收CAGR > 0% AND 潜力指标支撑
        → 最终类型 = 潜力成长股
    ELSE
        → 最终类型 = 成熟股

【优先级5-7：成长类型判断】
ELSE IF 行业属性 = "高成长赛道"
    IF 营收CAGR > 50% → 高成长股
    ELSE IF 营收CAGR > 20% → 成长股
    ELSE IF 营收CAGR > 0% → 候选"潜力成长股"（进入第三层）
    ELSE → 成熟股
ELSE（普通行业）
    IF 营收CAGR > 50% → 高成长股
    ELSE IF 营收CAGR > 15% → 成长股
    ELSE → 成熟股

【第三层：前瞻指标验证】（仅对候选"潜力成长股"）
IF 满足任一前瞻指标：
    - 在手订单/年收入 > 100%
    - 产能扩张计划 > 50%
    - 行业渗透率 < 20% 且公司为头部
    - 客户结构优化
    - 业务量增速 > 30%
    → 最终类型 = 潜力成长股
ELSE
    → 最终类型 = 成熟股
```

### 股票类型汇总（10种类型）

| 股票类型 | 识别条件 | 适用权重体系 |
| :--- | :--- | :--- |
| **早期成长股** | 高成长赛道起步期 + 增速>10% + 领头验证 | 早期成长股权重 |
| **早期转型股** | 转向起步期行业 + 新业务占比>3% + 增速>30% | 早期转型股权重 |
| **高成长股** | 营收增速>50% | 高成长股权重 |
| **成长股** | 营收增速20%-50%（高成长赛道）或 15%-50%（普通行业） | 成长股权重 |
| **潜力成长股** | 高成长赛道 + 营收增速0%-20% + 潜力指标支撑 | 潜力成长股权重 |
| **成熟股** | 增速<20%（高成长赛道）或 <15%（普通行业） | 成熟股权重 |
| **周期股** | 利润增速标准差 > 行业平均 × 2 | 周期股权重 |
| **亏损股** | 净利润（近3年平均）< 0 | 亏损股权重 |
| **转型股** | 满足转型股条件 | 转型股权重 |
| **潜力转型股** | 新业务占比3%-10%且增速>100% | 潜力转型股权重 |

### 识别依据记录

必须记录识别依据，包括：
- **第一层**：行业属性 + 行业阶段判断依据
- **第二层**：财务指标数据
- **第三层**：前瞻指标数据（如适用）
- **最终结论**：股票类型
