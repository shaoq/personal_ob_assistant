## 步骤 2：调用分析 Skill 生成报告
**操作内容**：

### 2.1 执行 Skill
执行命令：`equity-research:thesis <股票代码>(<股票名称>)`

示例：`equity-research:thesis 301292(海科新源)`

### 2.2 报告内容要求
除了 skill 自带内容外， **必须包含** 以下章节：

**A. 股票类型识别**
- 明确说明股票属于哪种类型（10种之一）
- 提供识别依据（参考阶段1步骤5的决策树）

**B. 投资建议-综合评级**

**B.1 五个维度评分**（每项 1-5 分）

| 维度 | 评分依据 | 加扣分 | 最终评分 |
| :--- | :--- | :--- | :--- |
| 行业前景 | 引用评分标准中的量化依据 | 列出加分/扣分因素 | 1-5分 |
| 公司竞争力 | 引用评分标准中的量化依据 | 列出加分/扣分因素 | 1-5分 |
| 成长性 | 引用评分标准中的量化依据 | 列出加分/扣分因素 | 1-5分 |
| 财务健康度 | 引用评分标准中的量化依据 | 列出加分/扣分因素 | 1-5分 |
| 估值合理性 | 引用评分标准中的量化依据 | 列出加分/扣分因素 | 1-5分 |

**B.2 综合评分计算**
```
1. 说明使用的权重体系（根据股票类型）
   示例："该股票为成长股，使用成长股权重：行业前景22%、公司竞争力22%、成长性30%、财务健康度13%、估值合理性13%"

2. 加权计算过程
   示例：
   基础综合评分 = 行业前景评分×权重 + 公司竞争力评分×权重 + ... = 3.65分

3. 成长系数判断（如适用）
   - 若适用成长系数，说明成长系数的确定依据和具体值
   - 若不适用，说明成长系数=1.00

4. 最终评分计算
   最终评分 = 基础综合评分 × 成长系数（上限5.0分）
   示例：3.65 × 1.10 = 4.02分
```

**B.3 综合评级**
- 根据评分范围对照评级定义给出对应评级
- 提供评级理由
- 若应用了成长系数（≠1.00），需说明成长系数的确定依据

**C. 投资逻辑**
核心投资逻辑的 3-5 点总结

**D. 投资论点总结**
看多/看空的关键论点归纳

### 2.3 表格样式规范
1. **表格对齐格式**：分隔行使用对齐的冒号格式
2. **列宽控制**：单列内容不超过 30 个中文字符
3. **避免**：单元格内换行、过长内容、嵌套表格
4. **复杂内容**：使用无序列表或子标题代替

**输出结果**：
- ✅ 报告内容生成完成
- ✅ 包含所有必需章节
- ✅ 评分计算过程清晰
- ✅ 已准备核心逻辑和操作建议内容
