## 一、行业阶段划分

> **版本**: 2026-03-08
> **用途**: 行业阶段判断依据，用于识别"早期成长股"和"早期转型股"

### 1.1 行业阶段多维度定义（四维度综合判断）

| 维度 | 起步期 | 成长期 | 成熟期 |
|:---|:---|:---|:---|
| **渗透率** | <5% | 5%-20% | >20% |
| **技术成熟度** | 实验室/小规模验证 | 规模化应用 | 稳定迭代 |
| **政策支持度** | 方向明确但执行框架未完善 | 完整产业政策体系 | 政策中性/收紧 |
| **市场规模** | <1000亿 | 1000亿-1万亿 | >1万亿 |

### 1.2 行业阶段判定规则

```
IF 渗透率<5% AND (技术处于验证期 OR 市场规模<1000亿)
    → 行业阶段 = "起步期"

ELSE IF 渗透率5%-20% OR (技术规模化应用 AND 政策体系完善)
    → 行业阶段 = "成长期"

ELSE IF 渗透率>20% OR 技术稳定迭代
    → 行业阶段 = "成熟期"
```

### 1.3 典型行业阶段示例

| 行业 | 行业阶段 | 渗透率 | 技术成熟度 | 市场规模 |
|:---|:---|:---|:---|:---|
| 人形机器人 | 起步期 | <1% | 小规模验证 | <500亿 |
| 量子计算 | 起步期 | <0.1% | 实验室/原型 | <100亿 |
| 低空经济/eVTOL | 起步期 | <1% | 适航取证中 | <500亿 |
| AI大模型 | 成长早期 | 5%-10% | 快速迭代 | 1000亿-5000亿 |
| 数据中心 | 成长中期 | 10%-20% | 成熟应用 | 5000亿-1万亿 |
| 半导体 | 成长中期 | 15%-25% | 成熟迭代 | >1万亿 |
| 新能源车 | 成长后期 | 25%-35% | 成熟应用 | >3万亿 |
| 传统汽车 | 成熟期 | >80% | 稳定迭代 | >5万亿 |

## 二、行业分类体系

### 2.1 高成长赛道定义

高成长赛道是指符合以下**任意两个条件**的行业：
- **市场规模CAGR > 15%**：行业处于快速增长期
- **国家战略支持**：列入国家"十四五"/"十五五"规划重点发展
- **技术快速迭代**：存在颠覆性技术突破或技术升级周期短

### 2.2 行业分类表

| 行业标签 | 行业CAGR | 政策支持 | 技术变革 | 典型行业 |
| :--- | :--- | :--- | :--- | :--- |
| **高成长赛道** | >15% | 国家战略支持 | 技术快速迭代 | AI、人形机器人、数据中心、量子计算、半导体、创新药、新能源 |
| **稳定成长赛道** | 5%-15% | 中性支持 | 技术稳定升级 | 消费电子、家电、汽车、食品饮料 |
| **成熟/衰退赛道** | <5% | 政策收紧 | 技术停滞 | 传统制造、地产链、煤炭、钢铁 |

## 四、行业识别规则

### 4.1 识别方法

采用**三层匹配机制**：

1. **精确匹配**：公司主营业务描述与高成长赛道子行业名称完全匹配
2. **关键字匹配**：公司主营业务描述包含子行业识别关键字
3. **关联匹配**：公司主营业务与高成长赛道高度相关（如供应链核心环节）

### 4.2 识别优先级

```
IF 主营业务匹配 "高成长赛道" 任一子行业
    → 行业属性 = "高成长赛道"
    → 记录匹配的子行业名称

ELSE IF 主营业务匹配 "稳定成长赛道"
    → 行业属性 = "稳定成长赛道"

ELSE
    → 行业属性 = "普通行业"
```

### 4.3 多业务线处理

若公司存在多条业务线：

| 场景 | 判断规则 | 示例 |
| :--- | :--- | :--- |
| 单一主业 | 按主业判断 | 纯AI公司 → 高成长赛道 |
| 主业+副业 | 按主业判断（营收占比>50%） | 70%汽车+20%机器人 → 汽车行业（稳定成长） |
| 双主业 | 若任一主业属高成长赛道，则整体标记为高成长赛道 | 50%AI+50%传统制造 → 高成长赛道 |
| 转型中 | 按新业务方向判断，但需标注"转型股" | 汽车→机器人（新业务15%）→ 高成长赛道（转型股） |

## 五、延伸业务识别规则

### 5.1 延伸业务 vs 转型业务

| 维度 | 延伸业务 | 转型业务 |
|:---|:---|:---|
| 主业地位 | 主业稳定，仍是核心 | 主业衰退，正在切换 |
| 业务占比 | < 10% | > 5%（转型股标准） |
| 战略意图 | 锦上添花 | 赛道切换 |
| 典型示例 | 中鼎股份（汽车橡胶龙头 + 液冷/机器人布局） | 传统制造转向半导体 |

### 5.2 延伸业务识别条件（需同时满足）

| 条件 | 要求 | 验证方法 |
|:---|:---|:---|
| 主业稳定 | 主营业务占比 > 50% | 查看年报业务结构 |
| 延伸初见成效 | 延伸业务营收占比 1% - 10% | 查看年报新业务披露 |
| 方向正确 | 延伸方向 ∈ 高成长赛道清单（7大类） | 匹配本文档第三章 |
| 战略明确 | 有明确战略布局和投入 | 查看年报、公告、投资者调研 |

### 5.3 延伸业务与转型业务的区分逻辑

```
【判断流程】
IF 新业务占比 < 1%
    → 不识别（太小，影响有限）
ELSE IF 新业务占比 1% - 10%
    IF 主业稳定（占比 > 50%）AND 主业增速 > 0%
        → 识别为「延伸业务」
    ELSE IF 主业衰退（占比下降 > 10个百分点 或 增速 < -10%）
        → 识别为「潜力转型股」（早期转型）
ELSE IF 新业务占比 > 10%
    → 按「转型股」识别规则处理
```

### 5.4 常见延伸业务方向表（按主业类型分类）

| 主业类型 | 常见延伸方向 | 判断依据 |
|:---|:---|:---|
| **汽车零部件** | 液冷热管理、人形机器人执行器、智能驾驶 | 技术同源，客户复用 |
| **消费电子** | AR/VR设备、汽车电子、智能家居 | 产业链延伸 |
| **传统制造** | 工业机器人、自动化设备、智能制造 | 技术升级 |
| **化工材料** | 锂电池材料、半导体材料、生物医药材料 | 材料应用拓展 |
| **通信设备** | 数据中心、算力基础设施、AI服务器 | 技术转型 |
| **医疗器械** | AI辅助诊断、机器人手术、脑机接口 | 技术融合 |

### 5.5 延伸业务数据获取位置

| 数据项 | 用途 | 数据来源 |
|:---|:---|:---|
| 延伸业务营收占比 | 识别条件判断 | P4（年报业务结构） |
| 延伸业务营收增速 | 成长性加分判断 | P4（年报同比数据） |
| 延伸业务所属行业 | 高成长赛道匹配 | P4（年报业务描述） |
| 延伸业务战略布局 | 战略明确验证 | P4（年报战略章节） |
| 延伸业务头部客户 | 行业前景加分 | P4（公告、调研纪要） |

## 六、使用说明

### 6.1 在分析报告中的应用

1. **步骤4（数据查询）**：查询公司所属行业
2. **步骤5（股票类型识别）**：
   - 根据行业名称匹配本配置文件
   - 确定行业属性（高成长赛道/稳定成长赛道/普通行业）
   - 应用到三层识别决策树

### 6.2 配置文件维护

- **更新频率**：每半年审查一次，根据行业发展趋势调整
- **新增行业**：当新行业满足高成长赛道定义条件时，可新增
- **调整依据**：行业CAGR变化、政策支持变化、技术突破
