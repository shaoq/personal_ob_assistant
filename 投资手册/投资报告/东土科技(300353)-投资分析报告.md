# 东土科技(300353)投资分析报告

## 一、公司概况

**北京东土科技股份有限公司**（股票代码：300353）成立于2000年，2012年在深交所创业板上市，是国内工业互联网底层根技术龙头。公司是国内唯一实现"工业以太网国际标准-硬实时操作系统-控制芯片"全栈自主可控的企业，深度绑定国家新型工业化战略。

| 项目 | 内容 |
|:-----|:-----|
| 股票代码 | 300353.SZ |
| 所属行业 | 工业互联网/工业软件 |
| 总市值 | 约126亿元（2026-03-20） |
| 主营业务 | 工业网络通信、工业操作系统及软件、工业AI控制器 |
| 核心产品 | AUTBUS总线、鸿道Intewell工业OS、工业AI控制器 |
| 战略方向 | 工业AI+具身机器人、国产替代、全栈自主可控 |

公司形成三大业务板块：工业网络通信（核心业务，2024H1收入2.73亿元）、工业操作系统及工业软件（2024年收入1.4亿元，毛利率66.36%）、大数据及网络服务。2024年已与半导体、机器人、数控机床、能源电力等20家企业签订规模化应用协议。

## 八、风险提示

### 行业风险
- 工业互联网/工业软件行业竞争加剧，华为等巨头加大投入
- 下游制造业资本开支波动影响需求

### 公司风险
- **盈利质量待验证**：扣非净利润2024年刚转正仅923万元，盈利基础薄弱
- **营收持续承压**：2024年营收同比下降11.52%，2022-2024年CAGR为负
- **商誉减值风险**：历史多次外延并购积累商誉，存在减值隐患
- **研发资本化率变动**：研发投入资本化率大幅变动影响利润表真实性
- **应收账款风险**：规模较大需关注回款质量

### 估值风险
- **PE极高**：当前PE 311倍，远超行业均值，动态PE约144倍仍严重高估
- 若工业AI控制器业务增速不及预期，估值面临大幅回调风险
