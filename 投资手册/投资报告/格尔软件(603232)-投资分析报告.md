# 格尔软件(603232) 投资分析报告

## 一、公司概览

### 1.1 基本信息

| 项目 | 内容 |
|------|------|
| 公司名称 | 格尔软件股份有限公司 |
| 股票代码 | 603232.SH |
| 上市日期 | 2017年4月 |
| 成立时间 | 1998年3月 |
| 注册资本 | 2.34亿元 |
| 总市值 | 约57亿元（2026年3月） |
| 所属行业 | 软件和信息技术服务业/商用密码 |
| 主营业务 | 以公钥基础设施(PKI)为核心的商用密码软件产品的研发、生产和销售及服务 |

### 1.2 公司定位

格尔软件是国内商用密码领域的骨干企业，是国内较早研制和推出PKI公钥基础设施产品的厂商。公司以密码技术为核心、以身份为中心，面向政务、金融、军工、企业提供数字信任和数据安全解决方案。深耕密码行业27年，拥有约700名员工，本科及以上学历占比约99.3%。

### 1.3 核心资质与荣誉

- 44余项发明专利、100余项计算机软件著作权
- 牵头或参与制定国家标准20余项、行业标准40余项
- 2次荣获国家科学技术进步奖二等奖
- 多次荣获上海市科学技术进步奖和党政密码科技进步奖
- 为32个国家部委、100多家国企央企、200多家银行提供服务
- 通过CMMI3认证、ISO9001质量认证、ISO27001信息安全管理认证

## 四、核心竞争力分析

### 4.1 技术优势

**抗量子密码技术领先**
- 自2018年起布局抗量子密码技术
- 全线密码产品已初步具备抗量子能力
- 国内首家通过中国信通院抗量子密码验证的厂商
- 参与国家抗量子密码标准制定
- 成功实现抗量子密码技术首次海外输出（非洲某国项目）

**密码技术积累深厚**
- 拥有近70项发明专利和180项软件著作权
- 密码算法实现技术处于国内领先水平
- 基于CPU密码模块的内生密码技术，实现硬件级安全

### 4.2 生态布局

- **鸿蒙生态**：首批入驻鸿蒙Next生态市场和应用商店的密码厂商
- **云生态**：密码服务平台、云密码机与阿里云、华为云深度集成
- **产业链**：牵头成立G60量子密码应用创新中心，与华为、阿里云共建生态

### 4.3 客户资源

- 政务、金融、军工等高壁垒领域深度布局
- 客户粘性强，替换成本高
- 不存在核心大客户流失风险

## 五、成长性分析

### 5.1 行业前景

**政策驱动**
- 《网络安全法》《数据安全法》《密码法》持续释放合规需求
- 《关键信息基础设施商用密码使用管理规定》推动抗量子密码迁移
- 《国家数据基础设施建设指引》推动数据可信流通需求
- 预计2025年国内商密市场规模将突破1,500亿元

**市场空间**
- 全球抗量子密码市场规模预计2030年超500亿美元
- 数据要素市场化催生数据安全新需求
- 车联网、物联网、AI安全等新兴领域增量市场

### 5.2 公司增长动力

1. **抗量子密码产业化**：定增募资1.6亿元用于抗量子密码技术产业化项目
2. **数据安全业务**：可信数据空间平台研发，中标上海数据交易所1.2亿元项目
3. **海外拓展**：承接非洲某国数字信任体系建设项目，打开"一带一路"市场
4. **新兴领域**：车联网安全、AI智能体安全、零信任架构等

### 5.3 风险因素

1. **业绩兑现风险**：抗量子密码大规模商用时间表不确定
2. **市场竞争加剧**：三未信安等对手在云密码服务领域发起价格战
3. **现金流压力**：经营活动现金流持续为负
4. **客户预算收紧**：政务、金融客户信息安全预算缩减

## 十、投资论点总结

### 核心论点

**短期（1年内）：观望**
- 2025年年报即将披露，亏损幅度可能超预期
- 需观察2026年一季度业绩修复情况
- 估值调整风险较大

**中期（1-3年）：关注拐点信号**
- 抗量子密码订单落地节奏
- 数据安全业务放量情况
- 海外项目拓展进展
- 现金流改善情况

**长期（3年以上）：战略价值可期**
- 量子计算威胁下，抗量子密码刚性需求确定
- 数据要素市场化推动数据安全市场扩容
- 公司技术积累深厚，具备长期竞争力

### 关键催化因素

1. **正向催化**
   - 抗量子密码国家标准正式出台
   - 重大订单落地（亿元级别）
   - 2026年业绩扭亏为盈
   - 海外项目拓展取得突破

2. **负向催化**
   - 年报亏损幅度超预期
   - 应收账款坏账计提增加
   - 行业竞争进一步加剧
   - 政策落地不及预期

## 十二、风险提示

## 十三、结论

格尔软件作为国内商用密码领域的老牌企业，在PKI密码技术、抗量子密码等领域具备深厚积累和技术领先优势。然而，公司2025年业绩由盈转亏、营收大幅下滑31.84%，短期基本面承压明显。当前估值极高（PE 646倍），已充分反映市场对未来成长性的预期，存在调整风险。

**综合建议：观望**。建议等待2025年年报落地、业绩拐点确认后再做投资决策。中长线投资者可在股价回调后关注抗量子密码产业化带来的布局机会。
