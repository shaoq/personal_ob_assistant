# 中微半导(688380) - 投资分析报告

## 一、公司概况

中微半导体（深圳）股份有限公司(688380)成立于2001年，2022年在科创板上市，是国内领先的以MCU为核心的数模混合信号芯片设计公司。公司专注于为智能控制器提供芯片级一站式整体解决方案，主要产品包括MCU、ASIC（高精度模拟、电源管理、通信交互、功率驱动等）、SoC等芯片以及功率器件。

**产品结构(2025)**：

| 产品类别 | 营收占比 | 说明 |
|---------|---------|------|
| 消费电子芯片 | 40.58% | 智能家居、可穿戴等 |
| 小家电控制芯片 | 30.66% | 白色家电控制板 |
| 大家电和工业控制芯片 | 25.25% | 工业自动化控制 |
| 汽车电子芯片 | 3.44% | 车规MCU(新增长极) |

## 七、投资逻辑

1. **国产MCU替代龙头**: MCU国产化率不足15%，公司作为国内主要MCU厂商之一，受益于国产替代大趋势
2. **汽车电子新增长极**: 车规MCU BAT32A系列已通过AEC-Q100认证并批量交付，切入汽车供应链，未来空间巨大
3. **产品结构优化**: 高毛利产品占比提升，毛利率从33%提升至37.9%，盈利能力持续改善
4. **R&D投入领先**: 研发占比14%，高于行业平均，拥有RISC-V架构布局，技术储备深厚
5. **财务结构健康**: 负债率仅10.8%，无有息负债，经营现金流充沛，fabless轻资产模式

## 八、风险提示

1. **估值过高**: PE TTM 107x显著高于行业平均，存在估值回调风险
2. **竞争加剧**: MCU中低端领域价格战持续，毛利率承压风险
3. **汽车MCU拓展不及预期**: 车规认证周期长，客户导入进度可能慢于预期
4. **供应链依赖**: 晶圆代工依赖中芯国际等，产能波动和价格上涨风险
5. **技术路线切换**: RISC-V对ARM架构的替代可能带来产品调整成本
