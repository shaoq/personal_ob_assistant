## 一、公司概况

### 1.1 基本信息

北京凯德石英股份有限公司(股票代码:920179)成立于1997年1月15日,是一家专注于石英仪器、石英管道、石英舟等石英玻璃制品的研发、生产和销售的高新技术企业,2022年3月4日在北京证券交易所上市。

**核心定位:**
- 国内唯一通过中芯国际12英寸晶圆产线认证的石英制品企业
- 半导体石英制品国产化领军者,打破美日企业垄断
- 国家级专精特新"小巨人"企业
- 2025年北京民营企业专精特新百强榜第六位

**核心团队:**
- 董事长张忠恕:70岁,石英加工领域50年经验,"石英工匠"精神引领者
- 总经理张凯轩:美国国籍,半导体行业资深人士
- 核心技术团队:平均行业经验超15年,研发人员占比约15%

**重大事项(2026年1月29日):**
江丰电子(300666.SZ)公告拟以现金收购凯德石英控制权,目前停牌筹划中。交易完成后,江丰电子将成为凯德石英控股股东。

## 二、行业分析

### 2.1 半导体石英制品:国产替代最核心的材料环节

**行业地位与需求:**

1. **不可替代的"芯片容器"**
   - 石英制品贯穿半导体晶圆制造70%以上核心制程
   - 应用于扩散、氧化、刻蚀、沉积、清洗、离子注入等高温核心工序
   - 产品性能直接决定晶圆良率与制程稳定性
   - 纯度要求99.99%以上,金属杂质含量<0.1ppm

2. **全球市场规模与格局**
   - 2025年全球半导体石英制品市场规模预计突破100亿美元
   - 中国贡献超40%产能,但国产化率不足20%(高端市场<10%)
   - 全球市场被德国贺利氏、日本东曹/信越、美国迈图等国际巨头垄断
   - 前5大厂商合计市占率约90%,内资企业处于追赶阶段

3. **供需缺口持续扩大**
   - 全球晶圆厂产能持续扩张,SEMI预测2025年产能增长6.6%
   - 国内长江存储、长鑫存储等存储龙头加速扩产
   - 先进制程(14nm以下)对高端石英制品需求增速超20%
   - 供给端:海外厂商扩产谨慎,2026年下半年才达产,供需缺口超50%

4. **价格持续上涨**
   - 2026年Q1半导体级高纯石英砂合约价环比涨30%+
   - 4N8/5N级进口价突破16万元/吨,较年初涨25%-30%
   - 高端石英部件(12英寸晶舟)价格较国内高出2-3倍

### 2.2 国产替代的政策红利

**核心驱动因素:**

1. **供应链自主可控战略**
   - 美国对华芯片设备加征关税,本土晶圆厂加速导入国产配件
   - 国家大基金三期重点投资设备和材料领域
   - 高纯石英矿列为国家战略性资源

2. **国内晶圆厂扩产潮**
   - 中芯京城、华虹无锡新厂等产线建设加速
   - 12英寸晶圆产能持续释放,拉动高端石英耗材需求
   - 国产化率目标:从当前15%提升至30%以上

3. **技术迭代与产品升级**
   - 先进制程(7nm/5nm)对石英纯度、精度要求呈指数级提升
   - 第三代半导体(碳化硅、氮化镓)发展带来新需求
   - 化合物半导体(磷化铟、砷化镓)在光模块、射频器件应用拓展

## 三、核心竞争力分析

### 3.1 技术壁垒与认证优势

**核心技术突破:**

1. **12英寸高端产品认证领先**
   - 2020年通过中芯国际12英寸石英晶舟认证,国内首家
   - 产品纯度达99.9999%(6N级),金属杂质含量<0.1ppm
   - 在中芯国际亦庄新厂(月产能10万片12英寸晶圆)供货占比达35%
   - 适配14纳米制程,晶圆良率从92%提升至97%

2. **核心工艺优势**
   - 超大尺寸封头技术:焊接变形率<0.01%,较同行提升3倍
   - 一体化磨削成型工艺:表面粗糙度≤0.1μm,达纳米级制程要求
   - 氢氧焰微焊接技术:焊接点强度达12MPa,远超行业8MPa标准
   - 0.1mm超薄石英环焊接:精度行业领先

3. **专利与标准制定**
   - 截至2025年6月末,专利总数98项,其中发明专利32项
   - 2025年2月获"化学气象沉积CVD外管加工工艺"发明专利
   - 2025年1月作为主要起草单位制定3项团体标准,掌握行业话语权
   - 专利密度达1.2项/亿元营收,远超国内同行0.5项的平均水平

4. **国际认证推进**
   - TEL(东京电子)认证进入现场审核阶段,已交付3款核心产品样品
   - 若2026年通过认证,将切入全球90亿元/年高端石英耗材市场
   - 正在推进应用材料(AMAT)、泛林半导体(LAM)认证

### 3.2 客户资源与订单稳定性

**深度绑定头部客户:**

1. **半导体客户:**
   - **中芯国际:** 12英寸产品稳定供货,14纳米制程批量供应
   - **北方华创:** 占营收41.63%,长期战略合作伙伴
   - **长江存储:** 已实现12英寸高端石英晶舟供货
   - **长鑫存储:** 送样认证中,有望成为核心供应商
   - **华虹半导体:** 8英寸产品稳定供应
   - **通美晶体:** 磷化铟/砷化镓产线用石英坩埚,上半年订单3200万元

2. **客户粘性极强:**
   - 半导体客户认证周期长达2-3年,一旦进入供应链合作关系稳固
   - 产品多为非标定制,深度绑定客户产线
   - 2025年上半年半导体业务收入占比95.15%,订单可见性强

3. **特殊领域独家供应:**
   - 为北斗卫星第三代导航芯片配套砷化镓专用石英坩埚
   - 通过-196℃至1200℃极端温变测试无开裂
   - 航天科技集团唯一国产合格供应商,已交付120套

### 3.3 产能布局与产业链延伸

**三大生产基地联动:**

1. **凯芯新材料(亦庄新工厂)**
   - 高端石英制品产业化项目实施主体
   - 2025年上半年营收2326.40万元,同比暴增203.88%
   - 12英寸石英晶舟产能利用率已达78%
   - 二期项目启动建设,计划新增12英寸石英管道产能15万件/年
   - 2026年二季度投产后,12英寸产品总产能将突破50万件/年

2. **凯德芯贝(沈阳子公司)**
   - 2025年上半年营收4314.39万元,净利润1747.69万元
   - "半导体精密配件研发生产基地"主体工程已完工
   - 预计2025年12月试生产,新增12英寸石英环产能20万件/年
   - 2025年12月入选辽宁省创新型中小企业

3. **凯美石英(与通美晶体合资)**
   - 半导体级大口径石英玻璃管和高纯石英砂项目
   - 2025年上半年营收1414.54万元,一期已投产
   - 解决上游原材料"卡脖子"问题,降低进口依赖
   - 实现产业链协同,原材料自给率将显著提升

### 3.4 商业航天与化合物半导体新增长点

**商业航天赛道:**

1. **磷化铟衬底配套:**
   - 为通美晶体提供磷化铟产线用石英坩埚
   - 磷化铟衬底因供需失衡价格暴涨50%(2英寸产品涨至1.2万元/片)
   - 产品耐温波动性控制在±2℃,较欧美标准(±5℃)提升30%
   - 上半年订单金额3200万元,同比增速超60%

2. **航天级产品:**
   - 北斗三号导航芯片配套石英坩埚,通过极端温变测试
   - 航天科技集团九院、中航光电供应链
   - 商业航天+半导体国产替代双重催化

**化合物半导体爆发:**

- 磷化铟、砷化镓在光模块(800G/1.6T)、射频器件应用快速拓展
- 凯德石英作为核心石英制品供应商直接受益
- 已新增三安光电、天岳先进等2家核心客户
- 预计全年化合物半导体收入占比将突破15%

## 四、财务分析

### 4.1 业绩下滑,短期承压

**2025年业绩快报数据:**

1. **营收小幅下滑:**
   - 营业收入2.83亿元,同比下降7.56%
   - 光伏太阳能行业用石英产品收入781万元,同比下降87.48%
   - 半导体集成电路行业收入有所增加,但不足以弥补光伏业务下滑

2. **净利润大幅下滑:**
   - 归母净利润2327.66万元,同比下降29.26%
   - 扣非净利润2267.26万元,同比下降29.46%
   - 基本每股收益0.31元,同比下降29.55%

3. **盈利能力下降:**
   - 毛利率约43%(2025年前三季度为43.82%)
   - 净利率8.23%,较2024年(10.74%)下降2.51pct
   - ROE 3.14%,同比-1.48pct

### 4.2 利润下滑原因分析

**三大核心因素:**

1. **收入下降导致毛利额下滑**
   - 光伏业务收入骤降87.48%,拖累整体营收
   - 半导体业务增长无法弥补光伏业务缺口

2. **增值税抵减政策变化**
   - 2024年同期新增固定资产较多,可抵扣进项税额多
   - 2025年享受增值税加计抵减政策涉及的进项税加计抵减金额减少
   - 对利润端产生负面影响

3. **存货跌价准备增加**
   - 部分库存商品市场需求减少(主要是光伏业务)
   - 2025年计提的跌价准备较上年同期增加
   - 存货/营收达167.04%,周转压力较大

### 4.3 现金流与财务健康度

**现金流改善:**

- 2025年前三季度经营活动现金流净额为正(具体数据未披露)
- 2025年Q1经营现金流净额-237.86万元,同比增加81.20%
- 货币资金/流动负债约30%,流动性压力需关注

**财务结构:**

- 资产负债率20.08%(2025年三季度末),低于行业平均31.95%
- 流动比率1.79,速动比率0.83
- 总资产10.63亿元,同比增长5.04%
- 归属上市公司股东每股净资产10.01元,同比增长2.67%

### 4.4 半导体业务持续增长

**2025年上半年亮点:**

- 半导体集成电路芯片用石英产品收入1.50亿元,同比+24.33%
- 占总营收比重95.15%,较2024年提升2.8个百分点
- 12英寸半导体业务收入同比实现高增长
- 凯芯新材料营收2326.40万元,同比+203.88%

## 五、江丰电子收购分析

- 目前停牌筹划中,预计停牌时间不超过5个交易日

### 5.2 收购动机与协同效应

**江丰电子收购动机:**

1. **产业链垂直整合**
   - 江丰电子主营高纯溅射靶材(金属材料)
   - 凯德石英专注石英制品(非金属材料)
   - 构建"金属靶材+非金属石英制品"全方位供应能力
   - 强化对半导体设备环节的配套能力

2. **产品矩阵拓展**
   - 江丰电子2025年上半年精密零部件业务营收4.59亿元,占比21.91%
   - 毛利率23.65%,而凯德石英毛利率约43-51%
   - 收购凯德石英可提升整体毛利率水平

3. **客户资源协同**
   - 两家公司均深度绑定中芯国际、华虹半导体等头部客户
   - 合并后可提供更完整的半导体材料解决方案
   - 增强客户粘性与整体议价能力

**协同效应:**

1. **技术协同:**
   - 双方在超高纯材料制备技术上的经验可相互借鉴
   - 加速新产品研发,如合成石英、碳化硅用石英部件

2. **市场协同:**
   - 江丰电子国际化布局助力凯德石英拓展海外市场
   - 共同推进TEL、AMAT、LAM等国际设备商认证

3. **供应链协同:**
   - 强化对中芯国际等头部客户的综合服务能力
   - 提升供应链安全与稳定性

- 2026年12英寸产品总产能将突破50万件/年,较2024年翻倍

### 6.2 看多论点

3. **产能释放:** 亦庄新工厂、凯德芯贝项目陆续投产,2026年产能翻倍
4. **客户优质:** 深度绑定中芯国际、北方华创、长江存储等头部客户,订单稳定
5. **收购催化:** 江丰电子拟收购控制权,带来估值重估与协同效应
6. **商业航天新增长点:** 磷化铟衬底配套、北斗卫星芯片供应,受益商业航天高景气
