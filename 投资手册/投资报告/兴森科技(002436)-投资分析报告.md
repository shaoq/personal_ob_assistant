### 主营业务描述

兴森科技是国内领先的PCB快速打样和小批量制造企业，近年来战略性进入半导体封装基板领域。公司业务分为两大板块：

1. **PCB业务**：提供PCB快速打样、小批量板制造服务，下游覆盖通信、工控、汽车电子、航空航天等领域，2024年收入43亿元，毛利率26.96%。
2. **半导体业务**：涵盖IC载板（含CSP封装基板和FCBGA封装基板），是公司战略转型的核心方向。FCBGA封装基板是高端芯片封装的关键载体，目前主要被海外厂商垄断，国产替代空间巨大。2024年半导体业务收入12.85亿元，同比增长18.27%，但受FCBGA产能建设影响，毛利率为-33.16%。

## 四、行业分析

### 行业概况

| 项目 | 内容 |
|------|------|
| 行业名称 | PCB及半导体封装基板 |
| PCB全球市场规模 | 约700亿美元 |
| IC载板全球市场规模 | 约150亿美元 |
| FCBGA封装基板市场规模 | 约80-100亿美元 |
| 行业CAGR | PCB 3-5%，FCBGA封装基板 15-25% |

### 行业发展阶段

- **PCB行业**：成熟期（全球增速3-5%，但高多层板、HDI板仍有结构性增长）
- **FCBGA封装基板行业**：成长期（AI算力需求爆发，先进封装占比快速提升，国产化率极低）

### 行业驱动因素

1. **AI算力需求**：GPU、HBM等高端芯片对FCBGA封装基板需求量大幅增长，供需缺口扩大
2. **先进封装趋势**：Chiplet、CoWoS等先进封装技术推动封装基板技术升级和价值量提升
3. **国产替代**：FCBGA封装基板被日韩台厂商（欣兴电子、IBIDEN、Shinko等）垄断，国产化率不足5%
4. **政策支持**：半导体产业链自主可控国家战略，封装基板是关键卡脖子环节

### 行业竞争格局

- **FCBGA全球**：被欣兴电子、IBIDEN、Shinko、UMTC等少数厂商垄断
- **国内竞争者**：兴森科技、深南电路、珠海越亚等少数企业布局
- **兴森科技地位**：国内FCBGA封装基板领先企业之一，广州基地建设中，年产能规划36万平米

## 八、投资逻辑

### 核心投资逻辑

1. **国产替代核心标的**：FCBGA封装基板是芯片封装的关键载体，被海外厂商垄断，国产化率不足5%，替代空间巨大
2. **AI算力需求拉动**：GPU、HBM等高端芯片对FCBGA封装基板需求爆发，全球供需缺口持续扩大
3. **产能领先优势**：广州FCBGA基地年产能36万平米，是国内少数具备大规模量产能力的企业
4. **PCB业务压舱石**：PCB业务年利润约6亿元，毛利率26.96%，为半导体业务转型提供稳定现金流

### 看空因素

1. **盈利尚未兑现**：半导体业务毛利率-33%，2025年全年净利润仅1.3-1.4亿元，对应260倍PE
2. **技术追赶风险**：FCBGA良率和产能利用率仍需提升，与国际龙头差距明显
3. **高资本开支**：FCBGA产线建设投入巨大，短期内财务压力较大
4. **行业竞争加剧**：深南电路等国内对手也在积极扩产，未来竞争可能加剧

## 九、风险提示

1. **盈利风险**：半导体业务持续亏损，若FCBGA产能爬坡不及预期，亏损可能扩大
2. **技术风险**：FCBGA封装基板技术壁垒极高，良率提升和量产进度存在不确定性
3. **估值风险**：当前市值371亿元但净利润仅1.3亿元，估值偏高，市场预期已较充分
4. **行业周期风险**：半导体行业存在周期波动，下游需求可能不及预期
5. **竞争风险**：国内外竞争对手扩产，未来行业竞争可能加剧，影响盈利能力
6. **客户集中风险**：若大客户订单减少或流失，将对业绩产生较大影响

## 十、投资论点总结

### 看多论点

- FCBGA封装基板国产替代空间巨大（国产化率不足5%）
- AI算力需求爆发，全球FCBGA供需缺口持续扩大
- 广州基地年产能36万平米，国内领先规模优势
- PCB业务提供稳定现金流（年利润约6亿元，毛利率27%）
- 2025年已实现扭亏为盈，经营现金流大幅改善（+199.53%）

### 看空论点

- 估值偏高（PE约260倍，PB 6.89，PS 5.43）
- 半导体业务持续亏损（毛利率-33%），短期拖累整体业绩
- FCBGA良率和产能利用率仍需提升，与国际龙头差距明显
- 高资本开支带来财务压力，资产负债率有上升风险
- 6个月涨幅66.29%，短期股价已有较大涨幅

### 操作建议

**观望**。兴森科技作为FCBGA封装基板国产替代核心标的，长期逻辑清晰，但当前估值偏高且半导体业务仍在亏损。建议关注以下催化信号：(1) FCBGA良率提升和毛利率转正；(2) 产能利用率明显改善；(3) 估值回调至更合理水平。待以上信号出现后再择机介入。
