# 奕东电子(301123)投资分析报告

## 一、公司概况

**奕东电子科技股份有限公司**（股票代码：301123）成立于1997年，主要从事FPC（柔性印刷电路板）、连接器及零组件、LED背光模组等精密电子零组件的研发、生产和销售。公司产品广泛应用于消费类电子、新能源汽车、储能、AI服务器等领域。2022年1月在深交所创业板上市。

| 项目 | 内容 |
|:-----|:-----|
| 股票代码 | 301123 |
| 所属行业 | 电子元器件/精密零组件 |
| 总市值 | 约138亿元（2025-03） |
| 主营业务 | FPC、连接器及零组件、LED背光模组 |
| 核心产品 | FPC、精密连接器、LED背光模组、液冷散热结构件 |
| 战略方向 | AI服务器液冷散热、PCIe Gen6/Gen7连接器 |

公司形成三大传统业务板块：FPC（2024年收入占比约37%）、连接器及零组件、LED背光模组。2024年总营收17.10亿元，同比增长16.37%。2024年起正式进军AI服务器液冷散热赛道，通过子公司可俐星实现批量出货，并收购深圳冠鼎51%股权（6120万元）强化液冷供应实力。

## 七、投资逻辑

1. **AI液冷转型方向明确**：2024年切入AI服务器液冷散热赛道，子公司可俐星2025年1-8月营收同比+688%，已进入全球头部客户供应链
2. **行业空间广阔**：智算中心液冷市场2024年184亿元，预计2029年突破1300亿元，CAGR约48%
3. **精密制造基础扎实**：20余年精密零组件制造经验，具备FPC+连接器+LED背光全制程能力
4. **AI服务器连接器受益**：配合头部大厂开发PCIe Gen6/Gen7连接器，AI服务器单台连接器用量超5000个
5. **盈利改善信号初现**：2025年前三季度扭亏为盈（净利0.29亿），营收同比+35%

## 八、风险提示

1. **持续亏损风险**：2024年净亏损4001万，2025年全年预测仍可能亏损（-6500万至-8500万），盈利拐点未确认
2. **毛利率持续下降**：3年下降约5个百分点（18%→12.95%），反映行业竞争加剧和定价权弱
3. **液冷转型不确定性**：液冷业务占比仍小（约3-4%），商业化进度、订单持续性存在不确定性
4. **应收账款风险**：应收账款周转天数130.84天偏慢，需关注回款质量和坏账风险
5. **原主业增长乏力**：FPC行业CAGR仅约6.76%，消费电子需求波动影响较大
6. **估值偏高风险**：138亿市值对亏损公司而言估值不低，若转型不及预期存在下行空间
7. **收购整合风险**：收购深圳冠鼎存在商誉减值和整合不及预期风险

## 九、投资论点总结

### 看多论点
- AI服务器液冷赛道高景气（CAGR约48%），市场空间巨大
- 液冷新业务爆发增长（+688%），已进入全球头部客户供应链
- 2025年前三季度营收+35%且扭亏为盈，经营改善信号初现
- 收购深圳冠鼎强化液冷产能，战略投入力度大
- 机构预测2026年净利润约3亿元，业绩弹性大

### 看空论点
- 公司仍处亏损状态，盈利拐点未确认
- 毛利率持续下降（3年降5pct），传统业务盈利能力弱
- 液冷业务占比仅约3-4%，对整体业绩贡献有限
- 138亿市值对亏损公司估值偏高
- FPC原主业竞争激烈，增长空间有限
- 应收账款周转慢，现金流质量需关注
