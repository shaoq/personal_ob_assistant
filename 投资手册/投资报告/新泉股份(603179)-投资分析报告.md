# 新泉股份(603179) 投资分析报告

## 一、公司概览

### 1.1 基本信息

| 项目 | 内容 |
|------|------|
| 公司名称 | 江苏新泉汽车饰件股份有限公司 |
| 股票代码 | 603179.SH |

| 总市值 | 约417亿元（2026年3月） |
| 所属行业 | 汽车-汽车零部件-汽车电子电气系统 |
| 实控人 | 唐志华、唐敖齐 |
| 主营业务 | 汽车饰件整体解决方案提供商 |

### 1.2 公司定位

新泉股份是国内领先的汽车饰件整体解决方案提供商，深耕汽车内饰行业二十余年。公司连续十八届入选全国百家优秀汽车零部件供应商，产品覆盖仪表板、门板、保险杠等全系列内外饰件，配套一汽解放、吉利、理想、特斯拉等主流车企。

### 1.3 发展历程

1. **创业深耕期（2001-2017年）**：从商用车内饰起家，逐步拓展至乘用车领域
2. **快速成长期（2017-2020年）**：上市后加速产能布局，进入国际知名电动车企供应链
3. **新能源爆发期（2021-2024年）**：绑定特斯拉、理想等头部客户，营收快速增长
4. **全球化拓展期（2025年至今）**：海外基地投产，切入座椅和机器人新赛道

## 四、核心竞争力分析

### 4.1 客户优势

**深度绑定头部车企**
- 特斯拉：Model Y仪表板总成核心供应商，已获Robotaxi座椅定点
- 理想汽车：L系列仪表板+门板+外饰全品类配套
- 比亚迪：海豹、海鸥等车型门板供应商
- 吉利、奇瑞：合作超15年，核心内饰供应商

**续单率高**
- 与核心客户续单率达95%
- 随客户成长，订单稳定性强

### 4.2 产品优势

**全品类内饰能力**
- 国内少数具备全品类内饰总成配套能力的供应商
- 从"单一部件"升级为"全屋定制"，平台化供应能力强
- 单车配套价值从3000元提升至万元以上

**技术壁垒**
- 掌握双料注塑、冷刀弱化等十多种先进工艺
- 国内最早搞定隐藏式气囊仪表板的厂家
- 轻量化产品减重15%以上

### 4.3 产能优势

**全球化布局**
- 国内：常州、上海、芜湖、宁波、长沙、成都、西安、长春等八大基地
- 海外：墨西哥（配套特斯拉北美）、斯洛伐克（配套大众欧洲）、德国（拓展欧洲市场）
- 马来西亚：2026年拟投资4200万美元设厂

**产能规模**
- 2025年总产能预计达500万套，较2023年翻倍
- 墨西哥工厂产能利用率有望提升至80%+

### 4.4 新业务布局

**座椅业务**
- 2025年4月收购安徽瑞琪70%股权，切入座椅领域
- 上半年座椅业务收入4344万元
- 已获特斯拉Robotaxi后排座椅定点，单车价值约8000元

**机器人业务**
- 2025年10月投资1亿元设立常州新泉智能机器人
- 开发执行器壳体、结构件产品
- 与特斯拉联动开发关节总成，初期产能10万套/年
- 与云深处、优必选等机器人公司联合研发

## 九、投资论点总结

### 核心论点

**短期（1年内）：业绩修复期**
- 行业年降压力逐步消化，Q4盈利能力有望环比改善
- 特斯拉Model YL、理想i8/i6等重点车型放量
- 关注毛利率修复节奏

**中期（1-3年）：成长兑现期**
- 海外工厂产能爬坡，海外收入占比有望提升至30%+
- 外饰+座椅业务放量，单车价值翻倍
- 机器人业务逐步贡献收入

**长期（3年以上）：全球化龙头**
- 成为全球汽车内饰龙头，海外市场份额持续提升
- 机器人业务打开第二增长曲线
- 长期价值取决于全球化进程与机器人业务落地

### 关键催化因素

1. **正向催化**
   - 特斯拉Model YL/Robotaxi销量超预期
   - 海外工厂产能利用率快速提升
   - 毛利率环比修复
   - 机器人业务获得大额订单

2. **负向催化**
   - 汽车行业销量不及预期
   - 年降压力持续加大
   - 海外工厂盈利不及预期
   - 机器人业务推进缓慢

## 十一、风险提示

1. **行业竞争风险**：汽车内饰行业竞争激烈，价格战可能挤压毛利率

## 十二、结论

新泉股份是国内汽车内饰龙头，深度绑定特斯拉、理想、比亚迪等头部车企，续单率达95%。公司全球化布局领先，墨西哥、斯洛伐克工厂投产，海外收入占比持续提升。从内饰延伸至外饰+座椅，单车配套价值翻倍。布局机器人执行器业务，打开第二增长曲线。

短期受行业年降压力影响，毛利率承压，利润端表现较弱。但公司长期成长逻辑清晰：新能源客户放量+海外市场突破+产品拓展+机器人新赛道。机构普遍预测2025-2027年归母净利润CAGR超25%。

当前估值偏高（PE 47倍），已反映部分成长预期。建议投资者在股价回调后逢低布局，重点关注海外工厂产能爬坡、毛利率修复、机器人业务进展等催化因素。

**综合建议：增持**。中长线投资者可关注75-80元区间的布局机会，跟踪业绩修复与成长兑现节奏。
