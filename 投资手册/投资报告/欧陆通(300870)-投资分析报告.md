# 欧陆通(300870)投资分析报告

## 一、公司概况

**深圳欧陆通电子股份有限公司**（股票代码：300870）成立于1996年，专注于电源领域，主要从事开关电源产品的研发、生产与销售。公司主要产品包括电源适配器、数据中心电源和其他电源，广泛应用于办公电子、网络通信、数据中心、电动工具及家用电器等领域。

| 项目 | 内容 |
|:-----|:-----|
| 股票代码 | 300870 |
| 所属行业 | 算力基础设施/服务器电源 |
| 总市值 | 约287亿元（2025-03） |
| 主营业务 | 开关电源研发、生产与销售 |
| 核心产品 | 电源适配器、数据中心电源、其他电源 |
| 战略方向 | 高功率服务器电源、液冷电源、第三代半导体 |

公司数据中心电源业务为长期发展战略重点，2024年实现营业收入14.59亿元（同比+79.95%），占总营收比重提升至38.4%。其中高功率数据中心电源业务实现营收7.80亿元，同比大幅增长536.88%，占数据中心电源业务收入比例提升至53.48%，保持强劲增长势头。

## 七、投资逻辑

1. **AI算力需求爆发**：全球AI大模型训练和推理推动服务器电源需求量价齐升，高功率电源渗透率快速提升
2. **国产替代加速**：台系厂商占据全球约45%份额，国内AIDC转向国产供应链趋势明确
3. **技术突破对标国际**：高功率服务器电源产品已比肩国际高端水平
4. **业绩加速增长**：数据中心电源收入+79.95%，高功率电源+536.88%
5. **产能扩张就绪**：东莞数据中心电源项目持续推进，订单饱满供不应求

## 八、风险提示

1. **估值偏高风险**：PE TTM 83倍、PEG 3.7，估值回调风险较大
2. **应收账款风险**：应收账款13.88亿元，周转天数约120天
3. **原材料价格波动**：电子元器件价格波动可能影响毛利率
4. **客户集中度风险**：服务器厂商客户集中度较高
5. **海外贸易摩擦**：海外业务存在贸易政策不确定性
6. **AI需求不及预期**：若全球AI投资放缓，高功率电源需求增长可能低于预期
7. **行业竞争加剧**：台系厂商和国内同行竞争激烈

## 九、投资论点总结

### 看多论点
- AI算力需求持续高增长，行业处于成长期
- 国内服务器电源国产替代趋势明确
- 高功率电源技术突破，产品竞争力强
- 业绩加速增长（2024年营收+32%，净利+37%）

- 估值显著偏高，PEG 3.7远超合理区间
- 应收账款规模大，资金占用风险
- 台系厂商竞争压力持续
- AI算力需求存在周期性波动风险
- 毛利率虽有改善但仍处中游水平
