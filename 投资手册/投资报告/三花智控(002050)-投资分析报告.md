# 三花智控(002050) 投资分析报告

## 一、公司概览

### 1.1 基本信息

| 项目 | 内容 |
|------|------|
| 公司名称 | 浙江三花智能控制股份有限公司 |
| 股票代码 | 002050.SZ |
| 上市日期 | 2005年6月 |
| 成立时间 | 1994年 |
| 注册资本 | 42.08亿元 |
| 总市值 | 约2052亿元（2026年3月） |
| 所属行业 | 通用设备/热管理/汽车零部件 |
| 实控人 | 张道才家族 |
| 主营业务 | 制冷空调电器零部件、汽车零部件及机器人执行器的研发、生产和销售 |

### 1.2 公司定位

三花智控是全球领先的热管理控制元器件制造商，制冷空调业务全球市占率超30%，汽车热管理业务国内领先。公司深耕热管理领域40余年，产品覆盖电子膨胀阀、四通换向阀、截止阀等核心零部件，客户涵盖格力、美的、大金、特斯拉、比亚迪等全球头部企业。2025年起，公司前瞻性布局人形机器人执行器赛道，已成为特斯拉Optimus旋转和线性执行器总成的核心供应商。

### 1.3 发展历程

1. **创业起步期（1994-2005年）**：从制冷配件起家，逐步建立四通换向阀等核心产品线
2. **上市扩张期（2005-2015年）**：深交所上市，产能持续扩张，全球市占率稳步提升
3. **新能源转型期（2015-2020年）**：切入新能源汽车热管理赛道，绑定特斯拉等头部客户
4. **机器人布局期（2020年至今）**：前瞻性布局人形机器人执行器，获特斯拉核心供应商资格

## 九、投资论点总结

### 核心论点

**短期（1年内）：业绩兑现期**
- 机器人执行器订单开始交付，业绩贡献逐步体现
- 新能源汽车热管理业务持续放量
- 关注Q1业绩验证与机器人订单执行进度

**中期（1-3年）：高速增长期**
- 特斯拉Optimus量产带动执行器订单大幅增长
- 机器人业务收入占比有望提升至25%+
- 在手订单持续兑现，业绩高增长确定性高

**长期（3年以上）：双赛道龙头**
- 热管理+机器人双赛道龙头地位稳固
- 人形机器人产业化加速，执行器市场规模持续扩大
- 长期价值取决于机器人业务的市占率与盈利能力

### 关键催化因素

1. **正向催化**
   - 特斯拉Optimus量产进度超预期
   - 机器人执行器订单持续增加
   - 季度业绩超预期
   - 新能源汽车客户销量超预期

2. **负向催化**
   - 机器人量产进度不及预期
   - 执行器良率提升不及预期
   - 行业竞争加剧
   - 汇率大幅波动

## 十一、风险提示

## 十二、结论

三花智控是全球热管理龙头，制冷配件市占率超50%，新能源汽车热管理国内领先。2025年公司前瞻性切入人形机器人执行器赛道，获特斯拉Optimus旋转和线性执行器总成核心供应商资格，12亿元大额订单锁定2026-2027年业绩。前三季度归母净利润同比增长40.85%，在手订单超500亿元，业绩确定性高。

当前估值偏高（PE 53倍），已充分反映市场对机器人业务未来成长性的预期。但考虑到公司热管理龙头地位稳固、机器人核心供应商卡位成功、在手订单饱满、财务结构健康，中长期成长逻辑清晰。

**综合建议：买入**。中长线投资者可关注48-50元区间的布局机会，跟踪机器人业务放量节奏与业绩兑现情况。在新能源汽车热管理+人形机器人执行器双轮驱动下，公司有望实现持续高速增长。
