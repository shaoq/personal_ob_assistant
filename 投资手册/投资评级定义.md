# 投资评级定义

## 评级体系说明

本评级体系基于五维度评分系统（行业前景、公司竞争力、成长性、财务健康度、估值合理性），综合评估后给出投资建议。

---

## 评级等级与定义

| 评级 | 预期收益 | 基准比较 | 风险程度 | 适合投资者 |
|------|----------|----------|----------|------------|
| **强力买入** | 显著跑赢大盘 | 超基准指数 15%-20%+ | 中高风险 | 激进型投资者 |
| **买入** | 看好 | 涨幅高于基准 10%-20% | 中等风险 | 积极型投资者 |
| **增持** | 谨慎推荐 | 小幅跑赢或优于大盘 5%-15% | 中低风险 | 稳健型投资者 |
| **观望** | 中性判断 | 与大盘基本持平（±5% 以内） | 低风险 | 保守型投资者 |
| **回避** | 看空 | 跑输大盘或下跌（跌幅超 5%-15%） | 高风险 | 不建议投资 |

---

## 评级与五维度评分对应关系

| 评级 | 综合评分范围 | 评分特征 |
|------|--------------|----------|
| **强力买入** | 4.5 - 5.0 分 | 五维度均优秀，行业景气度极高，公司竞争力突出，成长性强劲，财务健康，估值合理 |
| **买入** | 4.0 - 4.4 分 | 多数维度优秀，行业景气度高，公司竞争力强，成长性好，财务状况良好，估值相对合理 |
| **增持** | 3.5 - 3.9 分 | 整体表现良好，有一定亮点，但存在部分短板（如估值偏高或财务一般） |
| **观望** | 2.5 - 3.4 分 | 优缺点并存，行业或公司存在不确定性，建议等待更好入场时机或观望 |
| **回避** | 1.0 - 2.4 分 | 行业景气度差，公司竞争力弱，财务状况不佳，估值过高或存在重大风险 |

---

## 五维度评分标准

### 1. 行业前景（1-5分）
| 分数 | 标准 |
|------|------|
| 5分 | 行业处于爆发期，政策强力支持，市场空间巨大 |
| 4分 | 行业景气度高，政策支持明确，市场空间广阔 |
| 3分 | 行业平稳增长，政策中性，市场空间稳定 |
| 2分 | 行业增速放缓，政策收紧或市场竞争加剧 |
| 1分 | 行业衰退，政策限制，市场萎缩 |

### 2. 公司竞争力（1-5分）
| 分数 | 标准 |
|------|------|
| 5分 | 行业绝对龙头，技术壁垒极高，客户资源顶级 |
| 4分 | 行业领先，技术壁垒高，客户资源优质 |
| 3分 | 行业中游，有一定竞争力，客户资源一般 |
| 2分 | 行业下游，竞争力较弱，客户分散 |
| 1分 | 无明显竞争力，市场份额小 |

### 3. 成长性（1-5分）
| 分数 | 标准 |
|------|------|
| 5分 | 营收/利润增速>50%，订单爆发式增长 |
| 4分 | 营收/利润增速30%-50%，订单高速增长 |
| 3分 | 营收/利润增速10%-30%，订单稳定增长 |
| 2分 | 营收/利润增速<10%或下滑，订单疲软 |
| 1分 | 营收/利润大幅下滑，订单萎缩 |

### 4. 财务健康度（1-5分）
| 分数 | 标准 |
|------|------|
| 5分 | 负债率<30%，现金流充裕，应收账款健康 |
| 4分 | 负债率30%-50%，现金流良好，应收账款可控 |
| 3分 | 负债率50%-60%，现金流一般，应收账款偏高 |
| 2分 | 负债率60%-70%，现金流紧张，应收账款高企 |
| 1分 | 负债率>70%，现金流为负，应收账款严重 |

### 5. 估值合理性（1-5分）
| 分数 | 标准 |
|------|------|
| 5分 | PE/PB显著低于行业均值，PEG<1，安全边际充足 |
| 4分 | PE/PB接近或略低于行业均值，PEG 1-1.5，估值合理 |
| 3分 | PE/PB略高于行业均值，PEG 1.5-2，估值偏高但可接受 |
| 2分 | PE/PB显著高于行业均值，PEG 2-3，估值泡沫明显 |
| 1分 | PE/PB极高或无法估值，PEG>3，严重高估 |

---

## 使用说明

1. **评分方法**：对五个维度分别打分（1-5分），计算综合评分（五项平均）
2. **评级确定**：根据综合评分范围确定最终评级
3. **调整因素**：特殊情况下可适当调整（如重大利好/利空事件、行业拐点等）
4. **定期回顾**：建议每季度或重大事件后重新评估评级
